# 上下文系统 (Enterprise Context System)

## Definition

The enterprise context system is the accumulated, AI-readable representation of an organization's knowledge, decisions, experiences, and judgment calls — stored as both structured data and unstructured content (images, video, text, decision logs). In the GEA framework, context is the primary source of enterprise AI competitive advantage, because everyone can access the same foundation models but no one can replicate your context.

<div class="zh-trans">企业上下文系统是组织的知识、决策、经验和判断的累积性AI可读表示——以结构化数据和非结构化内容（图片、视频、文本、决策日志）的形式存储。在GEA框架中，上下文是企业AI竞争优势的主要来源，因为所有人都可以访问相同的基础模型，但没有人能复制你的上下文。</div>

## How It Works

**Two data types:**
1. **Structured data (Brown data)**: databases, tables — records of objective facts (sales figures, employee counts, expenses). This is the enterprise's "single source of truth."
2. **Unstructured data (Context data)**: images, video, meeting recordings, decision rationale, evaluation processes, taste judgments. In the pre-LLM era, this data was "dark" — unusable. Now it is the core asset.

**Self-learning mechanism:**
The context system uses AI to understand content's decision process, evaluation criteria, usage scenarios, and outcomes. It dynamically learns, becoming richer and more valuable over time. Jack Dorsey (Block founder): companies should urgently establish a unified context system as their enterprise world model.

<div class="zh-trans">两种数据类型：1.结构化数据（Brown数据）：数据库、表格，记录客观事实，是企业"唯一真相"；2.非结构化数据（Context数据）：图文视频、决策依据、评价过程、品味判断。大模型时代前这些数据是"暗数据"，现在是核心资产。自我学习机制：AI理解内容的决策过程、评价标准、使用场景和效果，动态自我学习，越积越厚越有价值。</div>

## Why It Matters

The key insight: in an AI Native world, every company uses the same foundation models. The competitive moat is not the model — it's the richness and depth of your enterprise context. A company that started building its context system in 2023 has an enormous compound advantage over one that starts in 2026, even if they use the same models.

<div class="zh-trans">关键洞见：在AI原生世界，每家公司使用相同的基础模型。竞争护城河不是模型——而是企业上下文的丰富度和深度。2023年就开始构建上下文系统的公司，比2026年才开始的公司有巨大的复利优势，即使两者使用相同的模型。</div>
