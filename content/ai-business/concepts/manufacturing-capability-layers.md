# 制造业四层能力框架

## 定义

制造业四层能力框架是吴琪（前罗兰·贝格中国区总裁）提出的制造企业竞争位置分析工具。核心观点：AI时代的企业竞争更多发生在"能力层"而不是"行业边界"上，企业的战略价值取决于在四个能力层中能占住哪一层。

## 运作方式

### 四层结构

```
┌──────────────────────────────────────┐
│  规则与标准层（最高价值/最难占据）        │
│  数据格式/接口协议/安全认证/生态规则     │
├──────────────────────────────────────┤
│  智能基础设施层                         │
│  工业软件/数字孪生/数据基础设施/行业模型  │
├──────────────────────────────────────┤
│  行业翻译层                             │
│  把通用AI翻译成行业可用解决方案的能力     │
├──────────────────────────────────────┤
│  物理执行层（中国当前优势所在）           │
│  制造/交付/工程化/大规模响应能力          │
└──────────────────────────────────────┘
```

**物理执行层**：制造、交付、工程化和大规模响应能力。中国在这一层的优势最强，是当前全球竞争地位的主要现实基础。但这一层的价值会随AI降低门槛而逐渐被追赶。

**行业翻译层**：把通用AI和通用算法翻译成某个行业可用解决方案的能力。决定了谁能把"通用智能"变成"行业生产力"。这一层是中国企业当前最大的升级机会窗口。

**智能基础设施层**：工业软件、数字孪生、数据基础设施和行业模型。掌握这一层，更接近制造业的"大脑"。西门子、PTC等欧美工业软件厂商长期占据此层。

**规则与标准层**：数据格式、接口协议、安全认证、模型评估与生态规则。谁在这一层形成影响力，谁就更接近未来产业生态中的高位。当前主要由欧美主导，是中国最需警惕的"规则性锁定"风险所在。

### 重要说明

这不是统一的价值阶梯——在特种材料、精密加工、航空发动机热端部件等领域，物理执行能力本身就可能是最高价值所在。框架的意义是帮助管理者判断：**在你所处的行业里，未来最关键的那一层到底在哪里。**

### 三国路径对比

| 国家 | 起点优势 | 典型升级路径 |
|------|---------|------------|
| 美国 | 基础模型/算力生态/平台能力 | 先占住智能核心，再向产业渗透 |
| 欧洲 | 工业软件/工程规范/制度治理 | 先工业体系和规则框架，再推广泛采用 |
| 中国 | 完整工业体系/超大规模场景/快速工程化 | 先把场景转化为行业智能能力，再反推平台/软件/标准上移 |

### 行业差异化能力上移战略

| 行业类型 | 代表行业 | 核心能力层重心 |
|---------|---------|--------------|
| 领先型 | 新能源汽车/消费电子/家电 | 物理执行层已强，需向行业翻译层+智能基础设施层突破，防止价值被平台截走 |
| 追赶中 | 高端医疗器械/半导体设备 | 先判断瓶颈在认知积累（AI可加速）还是物理底座（AI只是辅助） |
| 代际差距明显 | 先进制程/高端航空发动机 | 找AI可能带来非线性突破的切口，不要当万能钥匙 |
| 独特地位 | 稀土/特高压/高铁 | 从物理执行层升级为解决方案能力和规则影响力 |

## 为什么重要

四层能力框架解决了一个关键的战略误判：把"行业地位强"等同于"竞争优势可持续"。

中国制造业当前的优势主要在物理执行层，但AI正在使这一层的门槛降低、可复制性增强。西门子与英伟达2026年初扩大合作共建工业AI操作系统，Sandvik把AI嵌入制造软件+矿山设备+运维服务形成数字化解决方案层——全球领先企业争夺的已不再只是设备本身，而是设备之上的智能层与服务层。

**中国制造企业的战略窗口在于**：利用完整工业体系和超大规模场景（其他国家难以复制的数据训练优势），快速向行业翻译层和智能基础设施层上移，在规则性锁定、市场性锁定、基础设施性锁定三种机制关闭之前占据新的价值高地。
