# 智慧门店六维框架

## 定义

智慧门店六维框架是餐饮企业AI数字化转型的系统方法论，将门店智能化能力划分为从基础感知到决策智慧的六个维度：**看得见、听得见、能感知、会应变、善规划、会思考**。三层结构：基础层（感知）→ 分析层（响应与规划）→ 决策层（思考）。

## 六维能力详解

---

### 第一维：看得见（AI视觉）

通过摄像头+图像识别实时捕捉环境、物体、行为数据。

| 场景 | 能力 |
|------|------|
| 客流识别 | 辅助经营决策，提升营业利润 |
| 收银防弊 | 异常行为检测 |
| 产品外观识别 | 菜品品质管控，出餐优秀率管理 |
| 餐桌状态识别 | 翻台率管理，数餐盘 |
| 虫鼠/口罩识别 | 食品安全合规 |
| 夜间安防 | 智能警戒，降低安防成本 |
| 无人收货 | 收货流程自动化 |

**实际案例**：连锁水饺品牌，500+门店部署水饺品质检测（抓拍机→云端AI识别→小程序报告），识别准确率90%以上，菜品出餐优秀率**提升20%**。

**AI广告合规审核**（延伸应用）：
- 识别实地广告图场景 → 与模板海报对比 → 大模型判断海报内容/场景/要素是否一致
- 覆盖：活动现场物料 / 参与人数 / 产品堆头 / 终端广宣

---

### 第二维：听得见（语音AI）

语音识别+NLP，理解语音指令或对话内容，降低操作门槛。

**语音点餐**：
- 高精度语音识别（支持多语言/方言），智能语义理解（自动纠错/场景适配）
- 无缝对接ERP/CRM（数据实时同步），云端部署+边缘计算（低延迟）
- 价值：减少人工点餐时间，提升翻台率；适配全年龄段；用户偏好分析→精准营销

**会话智能（销售过程透明化）**：
- 采集来电/到访/门店销讲/跟客/签约全程数据
- ASR转写 + NLP处理 → 企业数据中台
- 应用：SOP监督 / 话术风控 / 员工画像 / 客户画像 / 商机识别 / 竞争力分析

---

### 第三维：能感知（IoT）

多模态传感器实时采集物理世界数据，构建对设备状态和环境的全面感知。

**覆盖设备**：空调/冰箱/保温柜/咖啡机/洗碗机/制冰机/炸炉/奶茶机/果汁机等

| 能力 | 价值 |
|------|------|
| 能耗监控 | 根据营业情况自动启停设备，降低水电费 |
| 食品安全监控 | 实时监控温度/水质/油质/微生物，超标预警 |
| 设备故障预警 | 异常自动报警报修，减少故障影响 |

**架构**：端（餐厅设备）→ 边（边缘网关）→ 云（IoT云平台 + 业务分析模型）→ 管（管理平台，巡检任务/故障上报）

---

### 第四维：会应变（实时响应）

基于实时数据动态调整策略，自动化控制或规则引擎快速响应业务变化。

**RPA+AI数字客服（分诊台模式）**：

```
用户输入
  ↓
AI意图识别（订餐/会员/门店信息/FAQ/投诉）
  ↓
在AI服务范围内 → AI直接处理（知识库调取）
不在范围内 → 推送门店专属客服人工处理
  ↓
AI实时质检（情绪异常/高风险 → 客服中心人工介入）
```

**分层知识库**（赛百味案例）：
- 门店信息库：地址/营业时间/电话/停车/母婴室
- FAQ库：活动流程/会员权益/发票/投诉
- 产品库：SKU/热量/套餐
- 业务流库：订餐/排号/开票操作步骤

**AI陪练**（员工培训）：
- AI模拟多角色（顾客投诉/政府来访/员工意外反应）
- 无疲劳/无情绪波动，24/7可用
- 价值：减少讲师岗位依赖，降低培训成本，提升技能掌握效率

---

### 第五维：善规划（预测优化）

AI算法分析历史数据，预判趋势/风险，优化资源配置。

**AI智能预估（四类变量）**：
| 变量类型 | 示例 |
|---------|------|
| 业务内部数据 | 销售记录/库存/促销/厨师排班 |
| 外部环境数据 | 天气/节假日/竞品动态 |
| 社会行为数据 | 社媒热度/舆情/大型赛事 |
| 高阶衍生变量 | "温差+周末"非线性影响/节假日前后7天波动 |

**应用场景**：
- **智能订补货**：支持未来30天半小时级别产品销量预测 → 精准采购计划，减少积压/缺货
- **AI智能排班**：运筹算法 × 半小时粒度 × 人员/技能/岗位SOP → 输出最优排班方案
- **智能预估效果**：备货准确率提升，损耗成本降低

---

### 第六维：会思考（决策智能）

大模型整合多源信息深度分析推理，从执行层向决策层升级。

**ChatBI（自然语言商业智能）**：
- 意图识别引擎 + 智能策略引擎 + 行业指标计算引擎
- 混合SQL生成（模板匹配+动态生成）
- 多轮对话，会话记忆，上下文追问
- 应用：经营分析 / 市场监控 / 决策汇报自动生成

**开店选址智能导航**：
- 营业额预测（历史数据+位置+商圈活跃度）
- 商圈分析（人口密度/消费能力/交通）
- 竞争格局评估（周边竞品/市场饱和度）
- 自动生成选址报告（租金区间/建议面积/预期ROI）

**合同智能审核**：
- 传统人工：3-4小时/份
- AI审核后：5-10分钟/份（**效率提升95%**，关键信息提取准确率95%，成本降低70%）
- 能力：结构化识别+风险标注（租金结构解析/合规风险）+附件多模态处理

**资讯雷达**（大宗采购助手）：
- 14品类全景监测（行情/政策/舆情/供应链风险）
- 智能知识问答（自然语言提问）+ 联网搜索 + 预警系统

## 为什么重要

餐饮行业的数字化路径与其他行业最大的区别：**门店是核心生产单元**，经营质量高度依赖现场的"人-货-场"协同。六维框架解决的核心矛盾是：如何将少数优秀店长的管理经验，通过AI"均一化"复制到数百乃至数千家门店——即"少数人的经验，多数人的能力"。
