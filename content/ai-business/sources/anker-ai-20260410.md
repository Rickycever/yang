# 安克创新 AI 转型四阶段实践

## Summary

A presentation by Anker Innovations' CTO (referred to as Neil/龚莹) at the same GEA forum, covering Anker's 4-phase AI transformation from 2023 to 2026. Anker is a ToC (consumer electronics) company, making this a concrete counterpoint to OTGO's B2B focus. The most striking data point: internal token consumption grew 100× in 2–3 months, from 10B/day to 120B/day.

<div class="zh-trans">安克创新CTO（Neil/龚莹）在同一GEA论坛的演讲，涵盖安克从2023年到2026年的四阶段AI转型。安克是ToC（消费电子）公司，与OTGO的B2B聚焦形成具体对照。最引人注目的数据：内部token消耗在2-3个月内增长了100倍，从每天100亿飙升至1200亿。</div>

## Key Points

**4-phase AI transformation:**
- **Phase 1 (2023)**: AI-assisted coding. Wrapped GitHub Copilot, required all R&D to use it. Result: 30%+ code output efficiency gain. Lesson: "see it to believe it" — show real results first.
- **Phase 2 (2024)**: Deep AI integration into business systems. Built AI Me (agent dev platform, 600–800 agents created) + WEILIAO (multimodal AIGC platform, 3M+ images/month, 50% quarterly growth).
- **Phase 3 (2025 H1)**: AI capability system building. Full org/process/data/talent adaptation. Introduced task→role→process agent progression model.
- **Phase 4 (2025 July+)**: "Alpha Moment" — token consumption grew 100× in 2–3 months. Full "business P-ification" (流程智能化).

<div class="zh-trans">四阶段AI转型：第一阶段（2023）：AI辅助编程，代码效率提升30%+；第二阶段（2024）：深度融入业务，AI Me（600-800个Agent）+微了（每月300万+张图，50%季度增长）；第三阶段（2025上半年）：AI能力体系建设，任务→角色→流程Agent演进模型；第四阶段（2025年7月起）："Alpha Moment"，token消耗2-3个月内增长100倍。</div>

**Key automation achievements:**
- Product detail page: from 1 month → 15–20 minutes via 5 agents working in parallel
- Platform API integration: from 2 weeks + 2 devs → 7 platforms/hour (AI reads API docs, writes code, deploys, tests)
- Advertising: 80% of ad content AI-generated; 20% fully automated end-to-end (including strategy and placement)

# 安克创新 AI 转型四阶段实践

## Summary

A presentation by Anker Innovations' CTO at the same GEA forum on 2026-04-10, covering Anker's 4-phase AI transformation from 2023 to 2026. Anker is a ToC consumer electronics company, making this a concrete counterpoint to OTGO's B2B focus. The most striking data: internal token consumption grew 100× in 2–3 months (10B/day → 120B/day). Monthly total: 2.5–3 trillion tokens.

<div class="zh-trans">安克创新CTO在同一GEA论坛（2026-04-10）的演讲，涵盖安克从2023年到2026年的四阶段AI转型。安克是ToC消费电子公司，与OTGO的B2B聚焦形成具体对照。最引人注目的数据：内部token消耗在2-3个月内增长100倍（每天100亿→1200亿），月均2.5-3万亿。</div>

## Key Points

**4-phase AI transformation timeline:**
- **Phase 1 (2023)**: AI-assisted coding via GitHub Copilot. All R&D required to use it. 30%+ code efficiency gain. Lesson: "see it to believe it" — demonstrate results first.
- **Phase 2 (2024)**: Deep AI integration. AI Me platform (600–800 agents created) + WEILIAO AIGC platform (3M+ images/month, 50% quarterly growth, 95%+ marketing materials generated).
- **Phase 3 (2025 H1)**: AI capability system. Full org/process/data/talent adaptation. Task→Role→Process agent progression model introduced.
- **Phase 4 (2025 July+)**: "Alpha Moment" — 100× token growth in 2–3 months. Full business P-ification (流程智能化).

<div class="zh-trans">四阶段AI转型：2023年AI辅助编程（代码效率+30%）→2024年深度融入（AI Me平台600-800个Agent，微了平台月均300万+图）→2025上半年AI能力体系建设→2025年7月起"Alpha Moment"（token消耗100倍增长）。</div>

**Key automation achievements:**
- Product detail pages: 1 month → 15–20 minutes (5 agents in parallel)
- Platform API integration: 2 weeks + 2 devs → 7 platforms per hour (AI reads API docs, writes code, deploys, tests with retry)
- Advertising: 80% of ad content AI-generated; 20% fully AI-managed (strategy + placement + reporting)
- Influencer marketing: AI auto-matches 2,000 influencers for new products and sends personalized outreach

<div class="zh-trans">关键自动化成就：商品详情页（1个月→15-20分钟，5个Agent并行）；平台API对接（2周+2人→每小时7个平台）；广告（80%内容AI生成，20%全流程自动化）；网红营销（AI自动匹配2000名目标网红并发送个性化邀约）。</div>

**Organizational transformation:**
- ADC model: 30+ 3–4 person squads (business expert + engineer + AI specialist), mapping SOPs and incubating reusable tools. Trained 300+ "seed talent" who understand both business and AI.
- UGS (User-Generated Software): business staff build their own systems with AI — Amazon operations team built their own promotion management and price monitoring systems.
- AI observability: internal system monitors every employee's AI usage (token spend, cost, output value), weekly feedback.
- "AI driving license" certification: mandatory for all staff; non-passers enter boot camp.

<div class="zh-trans">组织变革：ADC模式（30+个3-4人小队，业务专家+工程师+AI专员，已培养300+种子人才）；UGS（业务人员借助AI自行构建系统）；AI可观测体系（监控每位员工AI使用情况，每周反馈）；AI驾照认证（全员强制，未通过者入集训班）。</div>

**Model capability framing (L1–L5):**
L1: chatbot → L2: reasoning → L3: task execution (current mainstream) → L4: innovator (does things humans can't) → L5: investor/organizer (multiple agents forming swarm intelligence). Anker is actively pursuing L4–L5.

<div class="zh-trans">模型能力层级（L1-L5）：L1聊天机器人→L2推理→L3任务执行（当前主流）→L4创新者（能做人类无法完成的任务）→L5投资者/组织者（多个Agent形成蜂群效应）。安克正朝L4-L5迈进。</div>
