# AI时代战略落地与管理跃迁（CEIBS × 钉钉，2025-07-03）

## 摘要

钉钉在 2025 C10 圆桌派（杭州阿里巴巴西溪园区，2025-07-03）上发布的演讲 Deck。以阿里巴巴26年组织变革史为背景，提出 AI 时代战略落地方法论（DSTE框架），并介绍钉钉三大产品体系：**Agoal**（战略解码与OKR）、**TeamBition**（项目管理）、**组织大脑**（人才管理）。受众为 CEIBS 校友企业管理者/CEO。

## 核心要点

### Part 1：阿里巴巴组织变革历程

**底层逻辑框架**：使命/愿景/价值观 → 业务/组织/人 → 战略的实现离不开目标承接、组织人才保障、文化滋养三层支撑。

**五个阶段**：

| 阶段 | 时间 | 关键词 |
|------|------|--------|
| 初创 | 1999-2003 | 业务探索，扁平化组织 |
| 生长 | 2003-2008 | 1+1独立建制（淘宝分拆天猫） |
| 发展 | 2009-2015 | 技术驱动，中台萌芽（阿里云成立，淘宝一拆三） |
| 扩张 | 2015-2023 | 大中台战略→2023年1+6+N敏捷化变革 |
| 变革 | 2023至今 | 业务聚焦 + 经营责任制 + AI基建（通义/钉钉/闲鱼/夸克/1688四小龙）|

**核心洞察**："唯一不变的是拥抱变化" — 阿里从未固守某一组织形态，每次业务扩张或收缩都伴随组织结构调整。

---

### Part 2：阿里价值观体系

**价值观演进**（1999-2019）：

- 1999：可信、亲切、简单（自然生长）
- 2001：独孤九剑（互联网寒冬，生存导向）
- 2004：六脉神剑（蓬勃期，共创）→ 客户第一/团队合作/拥抱变化/诚信/激情/敬业
- 2012-2015：太阳&底线改革（价值观分层：业绩/价值观双维考核）
- 2019：新六脉神剑（数字经济体时代）

**五大传家宝**（文化行为器物化）：
1. **年陈文化** — 一年香/三年醇/五年陈实物奖励，增强归属感
2. **910客户日** — 全员接客服，"从客户中来，到客户中去"（逍遥子名言）
3. **510家人文化** — 510阿里日，关怀员工家庭，将家属纳入文化共同体
4. **信任文化** — "是毒草要长在阳光下"，开放简单的沟通氛围（内部论坛可议CTO任命/绩效复议）
5. **阿里家书** — 年度家书，记录奋斗历程，情感连接

**文化与价值观的区分**：
- 文化 = 言行举止，可见/可信（外显）
- 价值观 = 对重要度的排序，信念/主张/态度，决策依据（隐性）
- 警惕"文化两张皮"：说一套做一套

### Part 3：战略落地体系（DSTE）

**战略生成公式**：战略 = 未来 + 共识（"因为相信，所以看见"）

**四看三定（战略制定工具）**：
- 四看：看行业/看市场/看竞对/看客户
- 三定：定方向/定策略/定目标
- 钉钉 Agoal 结合专属大模型自动摘录竞情信息

**DSTE四阶段**：

| 阶段 | 工具 | 关键产出 |
|------|------|---------|
| 战略规划 | 钉闪会（闭门讨论） | 行业洞察、战略共识 |
| 战略解码 | Agoal OGSM一页纸 | 目标→策略→衡量标准→行动方案，公司→部门→子团队→个人四级拆解 |
| 战略执行与监控 | OKR对齐 + Teambition + 计分卡 | 事事有回应，件件有着落 |
| 战略评估 | 自评-环评-主管评三维绩效 | 战略闭环 |

**战略难落地的四大病因**：
1. 战略目标传递失真（上下没共识）
2. 无法敏捷响应市场变化（内外不适配）
3. 团队协作不顺畅（左右难合力）
4. 战略和执行两张皮（前后易脱节）

**数据**：只有不到10%的公司有效实施了战略规划的落地。

### Part 4：Agoal × TeamBition 产品体系

**Agoal**（AI战略目标管理）：
- 国内首个支持 OGSM 一页纸的战略解码平台
- 支持组织目标在公司→部门→子团队→员工个人四层拆解与可视化
- 计分卡：从指标库/经营合约/战略解码导入指标，按月更新，月/年视图
- 绩效评估：自评+环评+主管评，系统智能推荐环评人

**TeamBition**（钉钉项目管理）：
- 深度融合钉钉 IM，项目管理 IM 化（明确责任人/计划时间/交付成果）
- 五大能力：流程固化/在线计划/结构化任务协作/关键数据自动追踪/交付物管理（企业图书馆）
- 与 Agoal 目标关联：让目标不再"飘"，通过项目进展驱动 OKR 落地

### Part 5：组织大脑（人才管理体系）

**核心命题**：组织适配战略，人才适配组织

**七步人才管理链路**：

```
人才档案 → 人才标签 → 人才搜索 → 人才池 → 人才盘点 → 继任地图 → 数据罗盘
```

| 模块 | 核心能力 |
|------|---------|
| 人才档案 | 全员工作经历/绩效/技能/盘点结果一页纸，PC+移动端 |
| 人才标签 | 系统自动打标（985/高潜/续绩优秀等），支持自定义 |
| 人才搜索 | AI搜索引擎，自然语言条件匹配人才（如"今年晋升的人"） |
| 人才池 | 按规则自动入池（如高潜技术人才885人），支持人才对比 |
| 人才盘点 | 可量化盘点流程，多维9宫格（胜任力/学习力/可塑性），双端支持 |
| 继任地图 | 可视化组织设计与接班人规划（可立即继任/1年/1-2年可继任） |
| 数据罗盘 | 人才流动分析（入离职/转入转出），管理幅度与人效指标 |

### Part 6：AI时代的战略焦虑与解法

**FOMO效应**：企业在 AI 时代面临两种极端：
- 消极：盲目追技术热点，数据未ready，伦理风险
- 积极：战略先行，锚定 AI 长期价值，避免错过窗口期

**AI重构企业竞争优势的三大机会**：
1. 降低重复性工作投入，提升效率（自动化处理数据录入/客服常见问答）
2. 重构业务流程，提升顾客体验（推荐引擎/高价值客户识别）
3. 跳出信息茧房，挖掘隐藏信息（供应链价格波动预测/风险监控）

**钉钉"数字骨骼"隐喻**：
- 大脑 = 1号位战略远见（解析战略 SP/BP）
- 血液 = 客户价值创造流（解析分工 OKR/对齐关系）
- 肌肉 = 内部运营协作流（解析进展 周月报/项目进度；解析经验 知识库/复盘/绩效）

**阿里集团战略**（2025年）："用户为先，AI驱动"——2025.02.24宣布未来三年投入3800亿用于AI和计算基础设施（超过过去10年总和）。
