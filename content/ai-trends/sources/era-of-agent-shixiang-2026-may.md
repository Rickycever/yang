# The Era of Agent：拾象 AGI 投资洞察（2026-05）

## 摘要

AI 第一次开始系统性地加速自身发展，进入 2026 年后模型竞赛窗口从 6 个月压缩至以月/周为单位。Coding Agent 是科技史上增速最快的新物种，AI Coding ARR 预计 2026 年突破 1000 亿美元。报告分四个模块：Coding Agent 为何成为头部 Labs 的最强共识、AI Labs 的组织与战略文化竞争、Agent 时代 Playbook（To Human/To Agent 新地图）、硅谷前沿趋势（Robotics + Neo Labs）。

---

## 核心要点

### 01. Coding Agent 是当下最强共识

- Coding Agent 不是 Chat 的产品升级，而是已验证的新物种——如果说 Chat 是"头条"，Coding Agent 是"抖音"级机会
- Anthropic 的 Opus 4.5 是关键拐点：让用户真正从 chat 切换到 agent 模式，Anthropic 凭 Claude Code 预计年中追平 OpenAI ARR
- Coding 重要性：覆盖数字世界几乎所有任务；提升 AI Labs 自身研发速度；构建最短反馈回路（feedback loop 天然产生训练信号）；带来飞轮效应
- 做好 Coding 的壁垒不在技术 know-how，而在**组织与战略文化**：Anthropic 的核心是让数百名顶尖研究员愿意做"dirty work"（数据清洗）

### 02. AI Labs 组织与战略文化竞争

**Anthropic：战略聚焦的胜利**
- All in Coding 是在 C 端窗口关闭、Sonnet 3.5 获强反馈后摸索出的战略，非 Day 1 规划
- Top-down 决策 + 坚定 pre-training 信仰（不追 RL 神话）+ 面向 AGI 而非当下工作流设计产品
- AGI-native 文化：全员信奉 AGI 可实现，Dario 定期透明分享思考，人才稳定性显著高于同行
- 承诺兑现率极高，常超额完成

**OpenAI：被低估，但有结构性问题**
- ChatGPT WAU 接近 10 亿，付费用户超 5000 万，两次范式创新均由其引领
- 问题：长期重 C 端流量，对 Coding 优先级误判；约 300 个项目并行；0→1 文化强，1→10 产品打磨弱
- 优势：自下而上探索文化，1-2 名研究员押中方向可触发范式跃迁

**Google：最稳定的追随者**
- Gemini 3 带来增速，但已降至 13%；C 端/多模态优先导致 Coding 被忽视
- 底牌：TPU 资源极充裕（de-risk 集群超头部 Labs 训练集群）、Google Search/Android/Workspace 护城河
- 长期视角：即便 worst case 也能退守类 NVIDIA 的生态角色

**xAI：战略摇摆，已宣布解散**
- 2026年5月7日 Elon Musk 宣布 xAI 解散，SpaceXAI 与 Anthropic 签署算力合作（Colossus 1 超算服务）

### 03. Agent Playbook：新地图

- 旧坐标系 To B/To C 已失效，新坐标系：**To Human / To Agent**
- Agent 正成为独立消费者与生产者：Cloudflare 允许 Agent 自主开户、付费订阅、注册域名、部署代码
- Stripe 为 Agent 提供身份与支付中枢；Anthropic 通过 MCP/Skills/computer use 让 GUI 转为 CLI/API
- **Agent = Model + Harness**：Claude Code 的 Agent Loop 只有几十行，核心是"判断→工具调用→判断是否完成"的循环，让 Agent 从处理 1 分钟任务跨越到处理 2 小时以上任务
- Anthropic 商业模式转变：从按量收费 API 公司 → Agent 云公司/Agent OS；从卖 Token → 卖 Runtime

### 04. 硅谷前沿趋势

**Robotics：2026 年数据 Scaling 大年**
- 数据金字塔四层：① Egocentric（对应 pre-training）② UMI（对应 SFT）③ Teleop 真机（对应 SFT）④ World Model（对应 RL）
- 技术路线从 VLA 转向 World Action Model（WAM）：语言不是机器人的本质需求，物理智能才是；WAM 直接学视频不依赖 action-labeled data；泛化能力比 SOTA VLA 提升 2 倍以上
- 硬件优先：华人创业者的结构性机会，依托中国供应链理解打造机器人领域的 TSMC

**Neo Labs：两种路线**
- 路线 1：追寻下一范式（Continual Learning），代表：Jerry Tworek 的 Core Automation
- 路线 2：高价值 Vertical（芯片设计、AI for Science、高温超导），资本更偏好，与头部三家形成差异化
