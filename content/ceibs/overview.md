# 中欧商学院×明略科技 数字化营销

## 摘要

明略科技为 CEIBS（中欧国际商学院）提供数字化营销能力建设方案，聚焦 EMBA/MBA 招生全链路。核心逻辑：用 CDP+DMP 找到对的人，用 Leads 分级模型区分优先级，用企微 SCRM 驱动招生顾问高效跟进，用 MA 自动化培育尚未成熟的潜客。

## 核心概念

- [[ceibs/concepts/leads-grading-model|Leads 分级模型]] — Hot/Warm/Cold 三级分层，驱动招生顾问资源分配
- [[ceibs/concepts/lbs-targeting|LBS 定向投放]] — 基于目标场域（高尔夫/4S店/CBD）找高净值人群，转化率 +50%
- [[ceibs/concepts/rta-programmatic|RTA 程序化投放]] — 品牌自有数据实时决策竞价，精准 > 规模
- [[ceibs/concepts/oneid|OneID 打通]] — 跨触点统一用户视图，图计算 + Redis 实现

## 核心实体

- [[ceibs/entities/ceibs|中欧国际商学院（CEIBS）]] — 客户，中国顶级商学院，EMBA 学费 40 万+

## 关键数据

- 秒针×个推 Device ID → PN-ID 匹配率：**80%+**（行业基准 60%）
- LBS 定向 vs 媒体平台原生 LBS：转化率 **+50%**
- 明略三方标签体系：**1000+** 个标签，9 个类别
- 明略 CDP 数据治理效率：配置化 ETL，效率 **+50%**

## 时间线

- **2025.07**：明略科技提交数字化营销能力建设提案

## 三阶段路线

```
阶段一：SaaS DMP + 安全岛 → 验证精准投放效果
阶段二：CDP 建设 + Leads 分级模型 → 数据能力沉淀
阶段三：企微 + CDP 全流程打通 → 全链路自动化
```
