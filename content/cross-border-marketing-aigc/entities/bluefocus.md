# 蓝色光标 BlueFocus

## 概览

蓝色光标是头部综合营销集团，在 AIGC 出海营销中代表“集团级 All in AI”路径。其核心能力包括 BlueAI 平台、136 个 AI Agent、蓝色多脑盒、心影创作，以及用户画像和标注数据沉淀。

蓝标的路径强调战略级组织转型、数据资产和 Agent 矩阵，不适合普通中型公司直接复制，但可作为“数据飞轮 + 行业模型 + Agent 体系”的高配参考。

## 关键数据或事件

- 2024 年营收约 608 亿元。
- AI 业务收入从 2024 年约 12 亿元增长至 2025 Q3 约 24.7 亿元，长期目标进入百亿规模。
- 公开资料称其拥有 10 亿+ 用户画像、1.8 亿+ 标注数据、6000 亿+ API Token 调用量。
- BlueAI 结合自研模型与 Gemini、豆包、Adobe、通义千问、智谱等外部能力。
- 创意生产中，单条素材成本可降低 30-90%，月产超 5 万条创意。
