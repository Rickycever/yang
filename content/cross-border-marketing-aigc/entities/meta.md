# Meta

## 概览

Meta 是出海营销的主力广告平台之一，覆盖 Facebook、Instagram、Reels、Catalog、Partnership Ads 等广告形态。2025-2026 年，Meta 的核心变化是广告自动化程度继续提高，Andromeda、Advantage+ Creative 和 Advantage+ Framework 改变了自建工具的切入点。

对 AIGC 创意工具而言，Meta 既是分发平台，也是边界条件。平台已经免费化基础创意生成和自动组合能力，但并不开放广告表现到创意模型微调的完整闭环，也不可能做跨竞品平台的统一编排。

## 关键数据或事件

- Advantage+ 全家桶覆盖 400 万+ 广告主。
- Andromeda 提高创意在广告分发中的权重，精细受众包的边际价值下降。
- Advantage+ Creative 已内置图片变体、文本变体、扩图、动态叠加和图转视频等能力。
- 2025-10-08 起旧版 ASC/AAC API 不再支持新建，2026 Q1 停用，迁移至 Advantage+ Framework。
- CAPI 是与 Meta 算法进行稳定数据回传的核心通道，应与 Pixel 双轨部署。
