# 企业知识治理

## 定义

企业知识治理是指把企业内部文档、SOP、指标口径、业务经验、产品资料、活动复盘、导购话术和外部资料统一采集、清洗、切片、标注、检索、发布、审核和版本管理，使其成为企业 Agent 可调用的可信知识底座。

明略把它称为企业 AI 落地的“语料资产化”环节：先把隐性经验变成结构化知识资产，再让 Agent 在权限范围内检索、引用和执行。

## 运作方式

知识治理流程包括全渠道语料汇聚、知识清洗、智能切片、指标字典统一、向量化、多源知识接入、语义检索、关键词检索、结果排序、知识问答、标签管理、审核发布、权限设置、操作审计和效果评测。

应用场景包括品牌资产、市场洞察、公关传播、投放策略、互动营销、营销复盘、活动策划、私域运营、门店 SOP、导购话术、客诉处理和门店赋能。不同知识集合可以构成企业知识大脑、爆品策略大脑、智能营销大脑和门店赋能大脑。

## 为什么重要

没有知识治理，企业 Agent 只能基于通用模型回答，容易出现口径不一、幻觉、越权和无法复用经验的问题。知识治理把企业经验变成组织资产，并让总部标准向一线输出、一线实战经验向总部反哺。

商业上，企业知识治理的价值在于降低人员流失和组织扩张带来的经验损耗。它让企业从“靠人记住 SOP”转向“由系统维护 SOP”，再由 Agent 按角色、场景和权限进行调用。
