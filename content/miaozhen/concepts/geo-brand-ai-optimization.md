# GEO 品牌AI认知优化

## 定义

**GEO（Generative Engine Optimization，生成式引擎优化）** — 针对AI大模型（DeepSeek/豆包/Kimi等）的内容优化策略，通过优化品牌信息在AI生成内容中的可见度和推荐优先级，让品牌在AI搜索结果中被看见、被信任、被选择。

本质：AI时代的品牌信任建设工程，而非向AI"投毒"的流量把戏。

## 秒针GEO三阶段方法论

### 1. Capture 捕捉 — 捕捉用户搜索意图

关键问题：用户在AI上真正在搜什么？

方法：
- 从社媒大库/小红书/百度/抖音全量提取品类关键词
- 按战略一致性（品牌定位）+ 搜索热度双维度筛选
- 问题集搭建三法：5W1H法 / 用户旅程法（认知→兴趣→对比→决策）/ 痛点挖掘法
- 覆盖全购买决策链路（品类探索→车型对比→最终决策）

痛点解决："用户根本不会搜优化的关键词"→用搜索数据反推用户真实意图。

### 2. Evaluation 评估 — 评估品牌在AI上的可见度

**核心指标**：

| 指标 | 定义 |
|------|------|
| AI可见度 | 品牌被AI推荐的概率（出现次数/总提问次数×100%） |
| Top X推荐率 | 品牌作为TOP X推荐的比例 |
| AI引用率 | 品牌内容被AI引用的频率 |
| 产品优势维度覆盖度 | 对比类问题中本品优势维度出现率 |

**行业梯队基准（汽车）**：
- 第一梯队：≥80%
- 第二梯队：50-80%
- 第三梯队：20-50%
- 第四梯队：<20%

**统计可靠性要求**：每个关键词每个模型提问≥30次；城市差异无统计显著影响。

覆盖6大国内AI模型：DeepSeek / Kimi / 豆包 / 元宝 / 夸克 / 通义千问

### 3. Optimization 优化 — 优化AI对品牌的认知

**内容生产原则（SEA）**：

| 原则 | 方法 | 实证效果 |
|------|------|---------|
| Structuralization 结构化 | FAQ模块/长尾问题H2标题植入/"问题-证据-结论"逻辑链 | AI"易懂" |
| Expertise 专业性 | 引用权威来源（+30-40%引用率）/统计数据（+30-40%）/直接引文（+30%） | AI认为"懂行" |
| Authoritativeness 权威性 | 权重高信源植入/专家背书/获奖信息 | AI觉得"被认可" |

**各AI平台信源偏好**：

| 平台 | 主要信源偏好 |
|------|------------|
| DeepSeek | 行业权威网站/知名媒体 |
| 豆包（字节） | 头条号/抖音/抖音百科 |
| 元宝（腾讯） | 公众号/腾讯新闻 |
| Kimi | 知乎/搜狐/新浪（UGC+新闻） |

分发节奏：**启动期**（测试核心平台）→ **冲刺期**（集中放量）→ **稳定期**（长效维护）

## 正规GEO六大信任支柱

1. 精准理解用户意图
2. 决策过程透明可解释
3. 引用权威信源
4. 提供丰富数据实证
5. 允许交互式纠正与反馈
6. 依托平台自身权威性

## 实测案例：别克至境L7（17-25万增程轿车）

评估规模：8个关键词 × 3个模型（DeepSeek/豆包/元宝）× 10次 = 240次提问

| 评估结果 | 数据 |
|---------|------|
| 整体露出率 | 18.3%（第三梯队） |
| TOP3出现率 | 8.8% |
| DeepSeek表现 | 38.8%（最好） |
| 豆包 | 11.3% |
| 元宝 | 5.0% |
| 最强关键词 | 17-25万增程车推荐（63.3%） |
| 最弱关键词 | 17-25万空间大的车推荐（0.0%） |

**竞品格局**：比亚迪汉/汉L(40%) > 小米SU7(34.2%) > 凯美瑞(33.3%) >> 至境L7(18.8%)

优化策略：先做品牌露出（而非TOP占位），聚焦增程/性价比等强势词，空间/新能源类词需系统内容建设。
