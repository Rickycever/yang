# 白酒 DMP 人群三层策略

## 定义

白酒 DMP 人群三层策略是秒针为白酒品牌设计的受众分层框架，将目标人群分为**买酒人群**（存量精准触达）、**趋势人群**（增量破圈）、**品牌存量用户分析**（战略洞察）三层，分别对应不同的数据来源、规模预估与营销目标。

## 三层框架

### 第一层：买酒人群（约1,300万）

**核心逻辑：** 锁定有明确购买意图的用户，即"正在买酒"或"在买酒场所附近"的人群。

- **LBS定向**：全国烟酒行/商超酒柜周边地理围栏
- **标签叠加**：中高消费能力 + 酒类兴趣标签
- **应用场景**：精准竞价出价，转化率最优先

---

### 第二层：趋势人群（约700万—3,600万）

**核心逻辑：** 捕捉白酒消费的潜在增量群体，针对其场景特征定制内容。

| 子群体 | 规模预估 | 场景逻辑 |
|--------|---------|---------|
| 95后职场新人 | - | 初入职场，学习商务礼赠文化 |
| 海外华人/留学生 | - | 民族情感+传统文化向，异乡年节消费 |
| 宴请决策者 | - | 家宴/商宴主导者，高客单价场景 |

---

### 第三层：存量用户画像（战略分析）

**核心逻辑：** 通过CDP对接，分析现有品牌用户的真实画像，用于校准媒体投放策略。

以尖庄为例：
```
工薪阶层 35% + 蓝领 28% + 自由职业 25% + 银发族 12%
→ 调整媒体组合：下沉媒体权重 ↑，高端内容场景权重 ↓
```

## 数据支撑体系

| 能力 | 规格 |
|------|------|
| 移动设备覆盖 | 30亿+ |
| 日均数据交互 | 100亿+ |
| 线下POI | 6,000万+ |
| 标签体系 | 9类、3,000+ |

## CDP+DMP 集成路径

甲方第一方数据（CDP）通过**中立国**（隐私计算环境）与秒针 DMP 标签打通：
- PN（手机号哈希）→ DID（设备ID）转译
- 数据不出域，符合 PIPL 要求
- 实现第一方数据 × 第三方标签的双向增强

## 为什么重要

白酒行业存在特殊的数字化挑战：
1. **线下为主**：大量购买发生在烟酒行，无在线行为数据
2. **礼赠属性**：购买者≠饮用者，兴趣标签失效
3. **品类分层复杂**：高端/中端/大众人群重叠度低

三层策略通过 LBS + 行为标签 + 第一方画像回流，系统性解决上述问题。
