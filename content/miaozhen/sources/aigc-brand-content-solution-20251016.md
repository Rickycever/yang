## 背景：社媒重塑品牌叙事

**传播结构变迁**：长叙事（精英/权威/中心化/金字塔）→ 短叙事（大众/多元/去中心化/网状）

**算法逻辑两大特点**：
1. **喜新厌旧**：机器算法通过数据评估内容质量
2. **虹吸效应**：基于用户行为（时长/点赞/评论/收藏）决定是否放入更大流量池

**核心结论**：内容质量是社媒营销的关键——用户看到谁的内容，取决于内容质量。

---

## 短视频平台三池流量新逻辑

```
旧逻辑：自然流量池 + 广告流量池（两池）

新逻辑：
  内容流量池 — 平台目的：吸引用户、留住用户；商家关注：内容质量
  交易流量池 — 平台目的：高效转化好团品；商家关注：货盘质量
  广告流量池 — 平台目的：人/货/内容高效匹配；商家关注：广告效率
```

三池独立竞跑，各自有独立分发逻辑，品牌需要同时布局三池。

## 核心定位公式

> **高品质创意是内容为王时代的唯"一"性；AIGC是杠杆，投流是放大器**

产品核心交付：
1. **创意视频**（突破实拍能力边界）
2. **智能生成**（图片/文案规模化产出）
3. **海量铺陈**（KOC/KOS/素人矩阵分发）

## Content Flow：社媒内容AIGC工作流

### 洞察（AI Insight）
- 人群洞察：圈层 / 场景洞察 / 兴趣主题
- 内容洞察：热门话题 / 内容解码 / 内容画像

### 生成（AI Content Generation）
- **文案生成**：基于内容画像搭建工作流，批量去重文案
- **图像生成**：多角度产品图 / 实拍+AI生图模型训练，批量多样性产品场景图
- **视频生成**：视频脚本 / AI视频智能混剪 / AI视频素材 / AI创意视频
- KOC内容生成工具（Domo）

### 分发（Content Distribution）
- KOS系统：SAAS批量管理内容/发布任务/销售领取/账号发布/数据监测
- KOC分发：小红书 / 抖音 / 得物
- 素人营销：真人种草 / 活动造势 / 品牌造词

### 反馈（AI Feedback）
- 社媒/消费者反馈：沟通点 / 正负向 / 声量
- 效果反馈：曝光 / 互动 / 成本

## KOS/KOC 矩阵运营

**KOC**（Key Opinion Consumer 关键意见消费者）
- 中国网民11亿体量，高真实性和信任感
- AIGC赋能：内容策略brief → AI文案工具辅助 → 内容审核 → 数据回溯
- 管理平台：任务分发系统 + 内容登录系统 + AI审核 + 三天后数据回溯

**KOS**（Key Opinion Sales 关键意见员工/企业自有）
- 企业内部员工推广，比KOL更亲近垂直，比KOC更专业权威
- AI优化任务分配（智能分配给合适员工） + 内容生成（品牌调性适配各平台）
- 全链路管理：洞察产出→内容生成→内容分发→门店执行→效果监测→月度复盘
