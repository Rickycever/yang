## Data Agent 定义

Data Agent = **问题解决能力**（高）× **决策执行效率**（高）的数据专家型智能体。

| 象限 | 描述 |
|------|------|
| **Data Agent**（目标态） | 问题解决能力高 + 行动自动化高：深度业务理解、自动化执行与决策闭环、24/7不间断、零失误 |
| 通用Agent | 问题解决中 + 自动化高：创意发散场景可用，专业数据深度不足 |
| 大模型 | 问题解决中 + 自动化低：知识广泛，无法直接执行 |
| 传统数据产品 | 问题解决中 + 自动化低：专注预定义场景，缺乏业务语境理解 |

三大痛点解决：
1. **场景**：大模型缺乏用户粘性（缺乏业务闭环）
2. **行业**：通用Agent难以实现高效数据应用（没有Know-how）
3. **技术**：传统数据产品不够智能（业务使用门槛高）

## ML Data Agent 服务矩阵（四象限）

```
外部视角（面向客户）：
颠覆性AI（Native AI）    ←  现有业务AI赋能（Old Client × New Business）
产品服务新范式           ←  赋能现有研究服务

内部视角（面向运营）：
AI核心能力壁垒           ←  运营降本增效
（AI原料知识 + 新工作流）←  （员工提效工具）
```

四象限：
- **颠覆性AI**：Insight Flow为主体，原生AI（Native AI）模式，充分发挥AI优势
- **生产力AI**：运营提效，Gen AI工具赋能员工，基于内外部知识
- **赋能现有研究**：现有研究流程中AI赋能，提升效率与洞察深度
- **AI壁垒**：AI原料知识（与客户AI体系对接）+ 新工作流（新产品服务范式）

## DeepMiner 核心产品能力

### 三大交付价值
1. **流程可视化，决策透明化**：计划生成→报告输出全流程可追踪
2. **提升运营效率**：AI Agent自主完成 A-PDCA-R，自主校验与纠正
3. **整合企业级数据资产**：企业知识大融合，资源大协同

### 核心能力三维度

**① 多源数据整合**
- 秒针平台商业化数据（KOL/社媒舆情/媒体洞察/明日DMP）
- 客户一方数据（Excel文件上传或MCP接入）
- Web search + 主流电商/社交/市场研究/营销数据Agent服务
- 定制化数据提取Agent（指定权威机构/数据系统/社媒账号）

**② 专业自动化处理**
- 营销领域垂直大模型（品牌/意图关键词、PUGC体系标签）
- PLAN双模式机制（平衡创新探索与稳定交付）
- BUA + CUA数据抓取（自定义浏览器和计算机插件）
- 基于社交媒体9大行业知识图谱+场景洞察思维链

**③ 多Agent技术架构（Swarm+Supervisor）**
- Supervisor：统一理解任务 → 分解调度
- Swarm：多专业Agent高效协作执行
- Mano数据采集技术：大模型驱动的Browser User数据采集

## 三大企业级挑战与解决路径

| 痛点 | 根因 | DeepMiner解法 |
|------|------|--------------|
| 认知错位 | 业务人员看不懂数据，数据团队读不懂业务 | 自然语言交互，无需编码，多模型协同 |
| 执行脱节 | 数据洞察离业务落地隔着"最后一公里" | 任务分解+MCP调用，SQL/Python编码执行，深度研究报告生成 |
| 建设负担 | 技术资源消耗在重复数据开发，创新被边缘化 | 洞察思维链(CoT)模板化，Human-in-loop工作流 |

## 三大产品差异化能力

**1. 洞察思维链（CoT）沉淀 × 模板化**
- 将分析师经验沉淀为标准流程与模板
- 吸收"秒针系"经验：行业分析/营销复盘/人群洞察/趋势归因
- 分析师经验SOP化，复制能力，提升下限

**2. 数据系统连接 × 内外融合**
- 调用外部第三方系统（CDP/BI/数据平台/竞对监测）
- MCP集成企业私有数据（CRM/用户行为/投放数据）
- 路径：打通数据源 → 理解结构 → 生成洞察 → 输出报告

**3. Human-in-the-loop 分析工作流**
- Agent运行中途插入人工判断节点
- 多角色协同（跑数/建模/撰写/润色）
- 按需配置分析流程，支持灵活复杂需求

## 案例：宁德时代（CATL）

### 场景A：市场营销部 — 国内KOL & Social舆情分析
- **目标**：精准筛选新能源领域KOL，量化品牌在热点话题中的声量
- **数据基础**：Social魔方自动化采集实时多模态社媒数据
- **产出**：多维KOL评估（影响力/专业度/粉丝匹配度）+ 声量/情感/竞品排名
- **价值**：将复杂人工分析变为高效自动化流程

### 场景B：战略规划部 — 全球新能源电池市场需求预测
- **目标**：全球新能源电池需求/出货量/装机量预测，识别区域市场趋势与风险
- **数据基础**：抓取Research/GGII/中信建投等顶级机构结构化报告
- **产出**：交叉验证+推理验证的定量定性预测，短期风险与长期机遇预警
- **价值**：全球权威数据驱动战略决策，支持跨部门协同
