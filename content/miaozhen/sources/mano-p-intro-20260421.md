# Mano-P 1.0 GUI感知智能体模型产品介绍（2026）

明略科技发布的 GUI Agent 产品介绍，定位为"让 AI 打破只看不做的局限，直接在真实图形界面中跨平台执行复杂业务任务，打通自动化最后一公里"。Apache 2.0 开源，OSWorld 全球第一。

---

## 核心要点

**产品定位**
- 传统 RPA 依赖 API 或 HTML 解析，脆弱易失效，数据上云有合规风险
- Mano-P 纯视觉操控：无需 API，直接理解并操作任意 GUI 界面
- 完整闭环：感知 → 规划 → 操作 → 验证，四步一体化

**五大核心能力**
1. GUI 感知：纯视觉识别界面元素
2. 智能理解：将指令转成执行步骤
3. 任务规划：自动拆解复杂流程
4. 精准操作：点击输入切换都精准
5. 结果验证：自动校验并即时纠错

**性能指标（13项全球SOTA）**

| 指标 | 数值 | 说明 |
|------|------|------|
| OSWorld 任务成功率 | 58.2% | 全球第一（专有模型榜） |
| 领先第二名 opencua-72b | +13.2% | 绝对领先 |
| 权威基准榜单 SOTA | 13项 | 覆盖 ScreenSpot-V2、MMBench、UI-Vision、GUI Grounding、CUA、多模态感知、视频理解、长上下文 |

**三种部署形态**

| 形态 | 硬件要求 | 性能 | 适用场景 |
|------|---------|------|---------|
| 本地端侧 | Mac mini/MacBook M4，32GB+ | 4B模型476 tokens/s，峰值内存4.3GB | 高安全、离线环境 |
| 算力棒 | USB 4.0接口 | 即插即用 | 现有设备快速扩展 |
| 云端API | 72B完整版 | 最高性能 | 大规模、高并发 |

**对业务的三大核心价值**
1. 提升运营效率：7×24不间断，跨平台全自动执行，与 OpenClaw 生态无缝集成
2. 降低实施成本：无需 API 对接，Apache 2.0 开源可商业化，本地 Mac 即可部署
3. 保障数据安全：数据本地闭环，零上云，离线可运行，代码开源可审计

**7大行业场景**
- 智能对账（金融）：账单采集→规则匹配→差异提醒→自动汇总
- 合规填报（政务）：材料识别→字段抽取→表单生成→校验提交
- 电商运营（零售）：数据抓取→指标分析→任务分发→结果跟踪
- 合规监控（金融）：规则监测→风险识别→预警推送→处置留痕
- DevOps测试（IT）：环境检查→测试执行→结果分析→修复建议
- ERP迁移（企业IT）：数据盘点→映射转换→迁移执行→校验回滚
- 医疗理赔（医疗）：材料识别→规则核验→理赔计算→审批提交

共同特征：无API、不上云、人工成本高、跨系统。

**开源路线图**
- 已开源：Mano-CUA 核心技能（可接入 OpenClaw/Claude Code）
- 月内开源：本地模型 + SDK 组件
- 后续开源：训练方法 + 技术论文，Token 剪枝 + 混合精度量化，GUI-VLA 定制开发
