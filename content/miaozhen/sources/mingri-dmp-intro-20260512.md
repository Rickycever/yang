# 明日数据 DMP 产品介绍（2026-05-12）

## 摘要

明日数据（Tomorrow Data）2026年5月内部分享版产品介绍，共24页，覆盖两大主题：**产品升级概览**（DMP基础功能扩展、开放平台、分析平台、AI能力中心、数据市场）和**新数据源及案例**（京东电商行为、银联、百望车辆、航旅、消费品类）。核心战略方向：从传统数据管理平台向**API/MCP开放化 + AI Agent化**演进，并已在 ClawHub 上发布 DMP Agents。

---

## 核心要点

### 产品模块升级

**1. DMP 基础功能扩展**
- ID 圈包支持扩展：OAID/IDFA/IMEI、CAID、一方加密手机号
- 媒体直投新增：腾讯广告、支付宝灯火、巨量引擎/云图、B站/微博广告/华为鲸鸿、小红书灵犀/聚光、美图（in progress）
- 三方圈包接入：京东
- 新增能力：DSP查询（RTQ）、Look-alike、聚类分群、PN→DID/OM 转译

**2. 明日开放平台（Open Platform）**
- 提供 API + MCP 基础服务，打通 IT/系统集成与自动化能力
- 已覆盖标准 DMP 全功能 API 接口，总量近 30 个
- 已为部分客户提供系统化/AI化查询与服务
- 为 AI 构建基础环境，开拓新商务合作模式

**3. 明日分析平台**
- 面向分析师的自助式分析任务管理平台
- MTA 简单归因分析自动化：交付时间降至原来的 **25%**（1-2 个工作日）
- 标准化自动化，客户可按需自行执行

**4. AI 能力中心**
- **AI 市场（ClawHub）**：已在 ClawHub 发布 DMP Agents，1个月内 200+ 下载
  - 已实现：基础功能 Agents、离线画像 Agents、全自动画像洞察 Agents、智能人群圈包 Agents
  - 基础功能 Skills（开发中）
- **技能商店（Skill Store）**：按场景封装 DMP 专项 Skills，供 AI Agents 调用
- 画像助手 Agent：四层输出（总览 → 各标签 Top 分析 → 综合策略建议 → 深度分析建议）
- 智能圈包 Agent：四个人机协同节点（标签推荐确认 → 圈包方案确认 → 投前测量确认 → 投放订单确认）

**5. 数据市场**
- 已接入：京东标签圈包
- 计划接入：银联圈包/画像、友盟圈包
- 扩展数据源：电商、Social、O2O、交易（银联、百旺、航司、品类）

### 新数据源详情

**京东电商行为数据**
- 服务形式：维度圈包；支持 ID 类型：DID
- 覆盖：一级品类 57、二级 418、三级 3189，品牌/店铺/商品多维度
- 15+ 转化行为节点（浏览→搜索→收藏→加购→购买→售后）
- 策略人群：10大/15大靶群，与电商平台标签相似度 **70%+**
- 案例：竞品预购买用户 Social 再营销，拉新效率 +71%，新客赢得 23,000+；聚类放大包 New TA +90%，拉新效率 +200%

**银联数据**
- 服务形式：标签补充；支持 ID 类型：PN
- 维度：基础信息、持卡信息、交易消费（频次/总额/场景/时段）、风险信用评估、衍生指标
- 年龄段准确率：**74%**（vs 阿里对比，18-30 岁达 91%）

**百望数据（车辆信息）**
- 服务形式：标签补充；支持 ID 类型：PN
- 维度：VIN 码、品牌/车型/购车价格/购车时间/能源类型/新车/二手车
- 场景：车主档案补全、是否有车核验、车险到期、换车/卖车意向、二手车估价、车抵贷

**航旅数据**
- 服务形式：标签补充；支持 ID 类型：PN
- 维度：飞行次数/均价/折扣/里程、延误特征、舱位偏好、高频出发/到达城市

**消费品类数据**
- 服务形式：维度圈包；支持 ID 类型：DID
- 基于阿里系特定电商品类广告曝光行为的人群圈选
- 覆盖：生活服务、个护美妆、服饰、母婴宠物、家居、食品酒水、数码、交通工具、文娱教育、运动户外等全品类
