# 餐饮精准营销体系

明胜品智基于百胜中国8年数字化营销实战经验沉淀的精准营销方法论，核心是"公域拉新→私域留存→数据闭环"的三角体系，以RTA实时竞价投放为效果放大器，以CRM全生命周期管理为留存引擎。

---

## 核心框架：三层漏斗

```
品牌心智铺天盖地（Ads / IP / Social）
        ↓
GMV快速拉动（Digital Channel）
        ↓
生意基本盘稳步增长（LTV / Assets / Data）
```

- **第一层（品牌）**：KFS种草 / 精准触达 / 代言人 / 爆文 / 饱和触达
- **第二层（转化）**：各类数字渠道直接拉动GMV
- **第三层（复利）**：LTV提升 + 数据资产沉淀，推动生意基本盘的持续增长

---

## RTA精准投放

RTA（Real-Time Audience）是将企业一方数据实时接入平台竞价系统，对每次广告展示机会做人群判断，只对高价值人群出高价。

### 效果数据（半年18波项目平均）

| 人群策略 | vs 通投转化提升 |
|---------|--------------|
| 一方会员人群 | **+123倍** |
| 联邦学习人群 | +4.2倍+ |
| 算法人群 | +2.5倍+ |
| 兴趣人群 | +2.3倍+ |

- 潜在兴趣人群量级：超5亿
- 总体TA覆盖率：约39%

### YUMC 8年里程碑

| 时间 | 阶段/动作 |
|------|---------|
| 2018.12 | 初试程序化广告，RTA流量，标签沉淀 |
| 2019.10 | AdServing+三方标签，精准人群投放 |
| 2020.8 | DB3完整闭环能力，跨平台人群策略 |
| 2021.10 | 腾讯/百度/字节数据专区上线，联邦学习 |
| 2022.8 | 百胜媒介投放平台上线，全流程运营优化 |
| 2023.6 | CDP建设+标签运营持续深化 |

## 标签树体系

| 标签类型 | 示例 |
|---------|------|
| 主数据标签 | 注册信息/年龄段/会员等级/地理位置 |

## CRM全生命周期管理

以消费者生命周期为轴，对应五个阶段的运营动作：

| 阶段 | 核心动作 |
|------|---------|
| 引入期 | 会员开卡礼 / 首单专享 / 社群引导注册 |
| 成长期 | 定期优惠培养复购 / 积分 / 储值锁客 |
| 成熟期 | 新品活动 / 情感营销 / 客户关怀 |
| 衰退期 | 标签化管理沉睡客户 / 流失挽回模型 |
| 流失期 | 数据追踪 + 算法模型挽回高价值流失客户 |

## 私域触点体系

- **微信五件套**：公众号 / 社群 / 直播 / 朋友圈 / 1V1
- **自有渠道**：APP / 卡券 / 付费会员 / 储值积分
- **公域手段**：KFS（种草）/ 精准触达 / 代言人 / 爆文 / 饱和触达
