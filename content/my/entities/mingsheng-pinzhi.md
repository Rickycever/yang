# 明胜品智

## 概览

明胜品智（MY）是**明略科技 × 百胜中国**共同打造的连锁餐饮数字化解决方案公司。

| 维度 | 内容 |
|------|------|
| 愿景 | 成为餐饮智能的领跑者 |
| 使命 | 通过智能技术，让餐饮专注于美食 |
| 定位 | 连锁餐饮产业数字化的陪跑者 |
| 技术底座 | 数据智能&AI商业化（明略科技） |
| 行业经验底座 | 多元化管理理念和营运经验（百胜中国） |

## 股东背景

**明略科技**：
- 中国企业级数据智能应用软件提供商，深耕企业服务18年
- 服务超过2000家国内外头部企业、200余家政府机构
- 科技部"营销智能"国家新一代AI开放创新平台（第二批，2019年，继阿里云/百度/腾讯/科大讯飞/商汤之后）
- Gartner 10次上榜；吴文俊AI最高奖（2018、2020）；2000余项技术专利

**百胜中国（YUMC）**：
- 中国领先餐饮公司，1400+城镇，13000+餐厅
- 旗下品牌：肯德基 / 必胜客 / 塔可钟 / 小肥羊 / 黄记煌 / 拉瓦萨
- 过去7年战绩：+5000家门店，营收年均增长3%+，净利润增长10%，会员从8000万→4.6亿，员工保持40万

## 核心优势

| 维度 | 内容 |
|------|------|
| 规模复制 | 营运流程重塑和数字化系统生意驱动的落地和推广能力 |
| 行业经验 | 超过20,000家门店落地场景，连锁餐饮管理和高性价比技术 |
| 项目管理 | 资源整合式客户成功推动力 |
| 商业创新 | 数据智能、机器视觉、智能硬件的经济实用型探索 |

## 产品体系（全景）

四大平台：
- **全域营销服务平台**：企微社群/三方团购/抖音直播/媒体投放/会员中心/聚合外卖
- **全域业务平台**：商品/订单/支付/菜单/营销/发票中心
- **全域数据平台**：标签模型/报表计算/数据字典/货品中心
- **全域知AI平台**：决策式+生成式双引擎（推荐/预估/运筹/知识图谱/语音/视觉/文本）
