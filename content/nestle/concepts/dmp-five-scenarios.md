# DMP五大应用场景（雀巢Ad-DMP操作框架）

## 定义

雀巢中国 Ad-DMP 项目将数据管理平台的应用拆解为五个互相独立又可组合的操作场景，覆盖从媒介投放到电商转化再到会员运营的完整消费者旅程。每个场景有明确的目标、执行路径、核心团队和关键限制条件。

## 运作方式

### 场景一：Retargeting → 驱动互动/ROI提升

**目标**：对已触达用户做二次精准追投，提升媒介互动率或电商转化率

**执行逻辑**：
1. Ad-DMP 采集可监测回流的 DID 媒介用户数据（曝光+点击）
2. 按频次+行为拆分人群包（频次2-3次 / 4-6次 / 7次+ / 点击）
3. 站外Retargeting：促进品牌互动（核心：Marketing团队）
4. 站内/电商Retargeting：促进转化+新客（核心：EC团队）

**关键限制**：应用范围受媒体/广告形式是否支持 Nestle Ad-DMP 人群投放

**DMP赋能前后对比**：
- 之前：需额外付费给秒针，人群一整包，频次每活动1次
- 之后：BU无需额外付费，人群细化分包，每活动可Retargeting 2-3次

---

### 场景二：Look-alike → 扩展目标人群

**目标**：找到"还没买但很可能买"的品牌潜客，提升 CTR/媒介互动

**两条路径**：

**路径A（一方数据建模）**：以品牌购买/会员人群为正样本，曝光未购买为负样本，机器学习在 DMP 大库（1000+标签，20万+底层特征）建模，输出潜客人群包。Nestle DMP/京东/抖音三平台，每年共计15次建模。

**路径B（三方数据圈选）**：基于秒针三方数据按规则圈定潜客。核心定向维度组合：
- 人口学属性（性别/年龄/收入）
- 商品需求（替代品/关联品/竞品）
- 兴趣场景（健身/送礼/时尚/美妆等）
- LBS（商圈/区域）

**关键限制**：应用范围受媒体支持度；路径A效果依赖种子人群质量和放大倍数

---

### 场景三：Marketing to EC 评估（MET）→ 优化媒介策略

**目标**：量化 Marketing 对 EC 流量/互动/转化的真实贡献

**执行流程**：
1. DMP 准备媒介触达人群数据
2. 推送至电商分析平台
3. 输出：活动整体转化效果 + 分维度结果（媒体/频次/地域/人群标签/创意）
4. 同时评估：AIPL/4A 分布、消费者流转链路、活跃消费者资产构成

**服务参数**：活动结束后14天 + 7个工作日出结果；MTA每年6次

**关键限制**：可回传 DID 的媒体占比；该品牌电商销售占比

**DMP赋能前后对比**：
- 之前：基于整体趋势判断 Mkt 销售贡献，无法归因到具体媒体/频次/创意
- 之后：准确量化每个媒体/点位/频次/创意/人群的 EC 转化贡献

---

### 场景四：Consumer Analysis → 优化媒介策略

**目标**：深度洞察目标人群特征，输出媒介策略和创意方向

**数据来源组合**：
- Nestle Ad-DMP（媒体曝光/点击用户数据）
- Nestle CDP（会员/已购会员数据）
- 三方数据（外部供应商数据）

**输出维度**：人口属性、地域分布、设备偏好、媒介触点偏好、购买力分层

**典型洞察示例**（已有数据）：广东13%/北京11%/浙江9%；本科以上79%；安卓70%/苹果30%；爱奇艺-腾讯视频-芒果TV-优酷多平台重叠分析

---

### 场景五：CRM 运营增强 → 提升会员互动

**目标**：通过标签增补和流失预测，提升会员全生命周期互动与复购

**四大运营场景**：

| 运营场景 | 策略 | 核心标签需求 |
|---------|------|------------|
| 新客培育 | 个性化分组注册礼 | 性别/地区/来源标签 |
| 首次购买 | 基于行为个性化推荐 | 品类偏好/行为数据/内容偏好 |
| 复购促活 | 商品关联推荐/品类偏好 Campaign | RFM价值分群/订单数据/周期阶段 |
| 流失挽回 | 流失用户预测模型 + 个性化挽回 | 三方标签/行为数据/流失预测模型 |

**关键能力**：
- 为会员补充媒介曝光标签（通过 ID Mapping 打通 DMP 与 CRM）
- 流失会员预测模型（一方+三方标签建模）

---

## 场景组合逻辑

五大场景并非孤立，推荐的组合路径：
```
Consumer Analysis（场景四）→ 制定人群策略
    ↓
Look-alike（场景二）→ 找到潜客
    ↓
精准媒介投放触达
    ↓
Retargeting（场景一）→ 二次追投转化
    ↓
MET 评估（场景三）→ 量化效果/优化下一轮
    ↓
CRM 增强（场景五）→ 沉淀为会员资产
```

## 为什么重要

五大场景解决了雀巢广告的三大核心痛点：
1. **用户可留存**：45亿次曝光从数据黑洞变成可运营资产
2. **人群可运营**：从一刀切人群包到精细分层+持续沟通
3. **效果可评估**：从依赖媒体/代理商报告到自主量化 M2S 贡献
