## 摘要

秒针明略科技为雀巢婴儿营养事业部（NIN）制作的 DMP 项目提案，核心主张：秒针 DMP 具备母婴行业专属标签体系（BBV 关键词最高优先级）、跨全域媒体平台的数据覆盖（30亿+ 移动端月活设备）、以及合规的隐私计算接入能力（PSI + 中立计算区），能为雀巢奶粉业务提供从曝光到销售转化的全链路数据能力。

### 母婴行业专属标签体系

秒针为母婴行业定制了四类标签，**优先级从高到低**：

**1. BBV 关键词标签（最高优先级，项目核心差异点）**
- 基于用户在主流平台的搜索行为关键词建模
- 意图信号最强：主动搜索代表明确的购买意向
- 覆盖平台：百度、淘宝、天猫、微博、抖音等
- 对雀巢奶粉最具价值：搜索"奶粉推荐"/"海普诺钙"等关键词的用户直接对应转化意图

**2. 标准第三方标签**
- 基于设备行为、内容消费、线下 POI 建模
- 母婴人群通用：孕期判断、宝宝年龄段、育儿消费偏好

**3. 行为定制标签**
- 基于客户特定行为规则定制建模
- 例：在雀巢官微/APP 有互动行为的用户

**4. Social 关键词标签**
- 基于社媒内容（小红书、微博）的关键词提取
- 适合捕获种草行为，优先级低于搜索意图

---

### 合规架构：SaaS DMP + 隐私计算

秒针解决 PIPL 合规问题的方案：

```
品牌方 CRM/订单数据
         ↓
    隐私计算层（PSI + 中立计算区）
         ↓
  秒针 DMP 标签数据（不出境，不明文）
         ↓
    品牌 DMP 标签账户
         ↓
      媒体投放平台
```

- **PSI（Private Set Intersection）**：双方数据求交集，秒针看不到雀巢 ID，雀巢看不到秒针原始数据
- **中立计算区**：第三方可信环境执行计算，双方均不接触明文数据
- **数据不出境**：所有数据存储及计算在中国境内合规云

### DMP MCP Server 集成能力

秒针已具备 MCP Server 接口能力，支持 Agent 调用：
- 圈人工具：条件/自然语言 → 用户 ID 列表
- 标签查询工具：批量返回标签值
- 与 Agent-Ready 数据中台架构协同（Hermes Agent 框架可直接调用）

### 四大项目优势

1. **活数据（Active Data）**：实时更新的行为标签，非静态画像快照
2. **竞品营销监控**：能追踪目标用户看了哪些竞品广告（明细曝光监控）
3. **全域覆盖**：21 家媒体平台合规数据，7 家独家授权秒针
4. **广泛合作伙伴**：阿里妈妈、京东、字节等主要电商媒体均可对接打通

---

### 标签有效性四级评测体系

秒针对所有标签做四级评估，确保实际投放可用性：

| 评测层级 | 评估内容 |
|------|------|
| 供给合理性 | 标签人群量级是否充足，是否满足投放最低门槛 |
| 逻辑合理性 | 标签定义是否与目标人群行为逻辑一致 |
| 市场匹配度 | 标签人群与品牌已有客户画像的重合度 |
| 投放有效性 | 实际广告投放后 CTR/CVR 是否显著优于大盘 |
