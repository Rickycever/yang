# OpenClaw

## 概览

OpenClaw 是 NVIDIA 在 GTC 2026 发布的开放 Agent 协议/平台，定位为 Agentic AI 时代的基础协议层——类比 Agent 领域的 HTTP。NVIDIA 同步发布了 **NemoClaw**（OpenClaw 的参考实现），以及为 OpenClaw 优化的旗舰开放模型 **Nemotron 3 Super**。

---

## 关键数据与事件

- **发布时间**：GTC 2026 主题演讲
- **参考实现**：NVIDIA NemoClaw Reference OpenClaw
- **配套模型**：Nemotron 3 Super（最佳 OpenClaw 开放模型）
- **生态伙伴**：已有包括 OTGO 在内的合作伙伴接入

---

## 定位

OpenClaw 不是单一 Agent，而是定义 **Agent 如何发现、调用、协作**的开放标准。NVIDIA 提供：

1. **协议规范**：Agent 之间如何互操作
2. **NemoClaw**：构建专业化 Agent 的工具包（参考实现）
3. **Nemotron 3 Super**：专为 OpenClaw 任务优化的基础模型

**战略逻辑**：通过开放协议吸引生态，再以最优化的 NVIDIA 硬件（Rubin + NemoClaw + Nemotron）获取推理算力收益——类比 Android（开放系统）+ Google 服务（闭环变现）。
