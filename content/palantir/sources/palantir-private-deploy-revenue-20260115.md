# Palantir私有化部署持续收费策略

**来源类型：** 研究报告  
**生成时间：** 2026年1月15日  
**核心问题：** 私有化部署后如何避免 license fee 断崖下降？

---

## 摘要

深度解析 Palantir 及行业对标（Oracle、SAP）在私有化部署后如何保持持续收入。核心结论：Palantir 通过"订阅制 + 价值分享 + FDE驻场持续服务"三位一体组合，将私有化部署转变为持续增长的经常性收入，NDR 达到 134%。

---

## 核心要点

### 五大持续收费策略

**策略一：FDE驻场 — 持续价值发现引擎**
- FDE 团队长期驻扎，持续发现新用例，推动客户扩展使用深度与广度
- 扩展路径：单部门试点 → 验证ROI → 发现新用例 → 扩展至其他部门 → 企业级部署 → 新产品扩展
- 效果：前20大客户平均年支出从数百万增长至6400万美元

**策略二：价值分享 — 收入与客户价值深度绑定**
- 典型分润比例：20-30%（推断）
- 适用场景：供应链优化节省成本、运营效率提升（30-50%）、AI应用降本增效
- 优势对比：传统 license fee 收入上限固定，价值分享随客户价值增长而增长

**策略三：订阅制 — 持续服务而非一次性交付**
- 按用户数 / 数据量 / AI Agent数量计费，随业务增长灵活扩展
- 年度订阅 + 持续升级，非买断
- 3年总成本对比：传统买断 135-160%，Palantir订阅 100%，纯SaaS 120%

**策略四：分阶段扩展 — 从试点到全企业的自然增长**

| 阶段 | 部署范围 | 年度费用 | 累计增长 |
|------|----------|----------|----------|
| 试点 | 单部门 | 200-800万美元 | 基准 |
| 扩展 | 2-3个部门 | 500-1500万美元 | 2-3倍 |
| 企业级 | 全企业 | 2000-8000万美元 | 5-10倍 |
| 集团级 | 多企业 | 1亿美元+ | 10倍+ |

**策略五：产品组合扩展 — 持续创新推动增购**
- 产品演进路径：Gotham（2003）→ Foundry（2016）→ AIP（2023）
- 老客户增购逻辑：Gotham → Foundry（政府扩展到商业），Foundry → AIP（数据平台升级到AI）
- 效果：AIP 2025年商业市场增长 122%

### 成功案例

**Rio Tinto（矿业巨头）**：
- 从 Foundry 供应链优化（节省数十亿美元）→ 2024年11月续约并扩展 AIP 访问权限
- 验证了产品组合自然扩展路径

**DGSI（法国情报机构）**：
- 2025年12月续签3年合同，即使考虑主权替代方案，也需3年过渡期
- 验证了深度集成形成的技术护城河

**美国陆军 TITAN**：
- 100亿美元/10年合同，平均每年10亿美元
- 验证了持续服务模式（非一次性交付）对政府客户的适用性

### 行业对标

| 公司 | 策略 | 趋势 |
|------|------|------|
| Oracle | 分布式云：客户数据中心部署Oracle云基础设施，Palantir在OCI上运行 | 混合部署主流化 |
| SAP | 从On-Premise向订阅制转型，S/4HANA升级推动续费 | 订阅增长，License下降 |
| Palantir | 订阅 + 价值分享 + FDE驻场持续服务 | NDR 134%，持续超预期 |

### 关键成功要素总结

1. **持续价值创造机制**：FDE驻场 + 持续优化 + 新产品扩展，NDR 134%
2. **利益深度绑定**：价值分享模式让"客户成功 = Palantir成功"
3. **技术护城河**：Ontology 统一语义层 + Apollo 自动化部署 + FDE定制开发，替换成本极高
4. **分阶段降低风险**：POC 1-3个月 → 单部门 → 企业级，每阶段验证ROI
5. **产品组合协同**：Gotham → Foundry → AIP，老客户持续增购新产品
