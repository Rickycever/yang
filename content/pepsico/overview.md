# 百事×明略科技 Influencer Planner

## 摘要

明略科技为百事中国设计的端到端达人策略管理平台，打破各品类/品牌达人营销割裂状态。核心逻辑：用统一数据库沉淀达人资产，用 CCFI + 三级风险体系保障品牌安全，用 DeepMiner 热点自动化加速内容响应，用分钟级监测保障投后效果。

## 核心概念

- [[pepsico/concepts/influencer-planner-platform|Influencer Planner 平台]] — 四模块端到端平台（Planning/Vetting/Content/Measurement）
- [[pepsico/concepts/kol-vetting|达人审核与 CCFI 框架]] — 三级风险评级（红/黄/绿）+ CCFI 综合评分（Content/Cost/Follower/Impact）

## 核心实体

- [[pepsico/entities/pepsico|百事中国（PepsiCo China）]] — 客户，多品牌食品饮料集团

## 关键数据

- 达人数据库：小红书 **80W+**、抖音 **120W+**、快手 **50W+**
- 热点响应：**40秒**抓取 → **6分钟**生成完整 Brief
- 人工审核压缩：**70%+**（Vetting 自动化）
- SLA：**30分钟**响应 / **4小时**出方案 / **24小时**修复

## 三年路线图

```
Year 1（流程整合）：统一达人数据库，Vetting 自动化
Year 2（智能化升级）：Smart Brief 常态化，CCFI 全量覆盖，多平台实时监测
Year 3（生态延伸）：向代理商开放，与 Global 平台互通，Retail Media 联动
```

## 24 周实施计划

- W1-4：平台搭建 + 达人库数据接入
- W5-12：Planning + Vetting 模块上线（Phase 1）
- W13-20：Content Hub + DeepMiner 集成（Phase 2）
- W21-24：Measurement 全链路打通 + 验收
