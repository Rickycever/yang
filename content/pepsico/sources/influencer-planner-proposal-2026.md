# 百事×明略科技 Influencer Planner 项目提案（2026.01）

## 来源信息

- **文件**：Y2026 PEP Influencer Planner 项目提案_20260120_V2.pdf
- **提交方**：明略科技（秒针系）
- **甲方**：百事中国（PepsiCo China）
- **日期**：2026-01-20

## 背景与问题

百事中国达人营销面临四大核心挑战：

1. **策略割裂**：各品类/品牌各自为战，无法共享达人资产和投放经验
2. **筛选效率低**：人工审核达人耗时，无法在短时间内完成全面舆情排查
3. **内容生产慢**：Brief 制定、热点追踪靠人工，响应速度慢
4. **投后评估滞后**：缺乏实时监测，无法快速发现和处置风险内容

## Influencer Planner 平台四大模块

### 模块一：Planning（达人策略规划）

历史数据复刻 + 内核×外壳迭代框架：
- **历史数据复刻**：基于过往项目数据（平台、内容类型、达人层级、预算分配），生成最优组合参考方案
- **内核×外壳框架**：内核 = 核心传播主题（品牌调性，较稳定）；外壳 = 创意执行形式（随热点迭代）
- 输出：Smart Brief，包含达人画像要求、内容方向、投放节奏、预算参考

### 模块二：Vetting（达人审核）

三级风险评级体系（红/黄/绿）：
- **绿灯**：合规可用，进入候选池
- **黄灯**：存在潜在风险（历史争议/违规记录），需人工复核
- **红灯**：高风险，直接排除

审核维度：3年以上社交媒体舆情、品牌合作历史、平台违规记录、内容质量、粉丝真实性（反水军检测）

**CCFI 评估框架**（KOL综合评分）：
- **C** — Content Quality（内容质量）：互动率、内容原创度、垂类专业度
- **C** — Cost-Effectiveness（性价比）：CPE vs. 行业基准
- **F** — Follower（粉丝质量）：真实粉丝占比、粉丝画像匹配度
- **I** — Impact（传播影响力）：曝光量级、二次传播率

达人数据库规模：小红书 80W+、抖音 120W+、快手 50W+

### 模块三：Content（内容生产）

核心工具：Content Hub + DeepMiner 热点自动化
- **热点监测**：实时追踪平台热榜、社交舆情、竞品动态
- **自动化流水线**：40秒抓取热点 → 6分钟生成完整 Brief（含参考素材、关键词、发布时机建议）
- **多模态打标**：视频/图文内容 → 多模态特征提取 → embedding → LLM → 结构化标签（品类/场景/人群/情绪/调性）
- **Partner 协作**：破圈（小红书ISP）+ 极创美奥（抖音官方认证）+ 零壹贰叁科技（AIGC内容生产）

### 模块四：Measurement（效果测量）

- **分钟级实时监测**：内容上线后，每分钟更新曝光/互动数据
- **风险预警**：负面舆情自动触发告警（分级：红/橙/黄）
- **投后复盘报告**：达人表现评分、内容质量评估、ROI拆解、下次策略建议
- **SLA 保障**：30分钟响应 / 4小时出方案 / 24小时修复

## 合作伙伴生态

| 合作方 | 角色 | 能力 |
|--------|------|------|
| 北京破圈 | 小红书 ISP | 小红书官方服务商，达人资源+数据 |
| 极创美奥 | 抖音官方认证 | 抖音达人合作+流量资源 |
| 零壹贰叁科技 | AIGC 内容生产 | AI 短视频/图文批量生产 |
| DeepMiner | AI 智能体 | 热点分析、Brief自动化 |
| DOMO 多谋 | 内容策略引擎 | 千人千言内容矩阵（见 DOMO 相关页面）|

## 技术架构

**智能化多模态打标流程**：
```
原始内容（视频/图文）
→ 多模态特征提取（视觉+文本）
→ embedding 向量化
→ LLM 结构化理解
→ 标签输出（品类/场景/人群/情绪/调性）
→ 写入达人画像数据库
```

## 实施计划

**24 周实施路径**：
- W1-4：平台搭建 + 数据接入（达人库初始化）
- W5-12：Planning + Vetting 模块上线（Phase 1）
- W13-20：Content Hub + DeepMiner 集成（Phase 2）
- W21-24：Measurement 全链路打通 + 验收

## 三年路线图

```
Year 1：流程整合
  - 统一达人数据库（替代各BU分散管理）
  - Vetting 自动化（减少人工审核 70%+）

Year 2：智能化升级
  - Smart Brief 自动生成常态化
  - CCFI 评分体系全量覆盖
  - 多平台实时监测打通

Year 3：生态延伸
  - 开放给百事核心代理商共用
  - 与百事 Global 平台数据互通
  - 探索 Retail Media + Influencer 联动
```
