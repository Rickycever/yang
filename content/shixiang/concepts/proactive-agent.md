# Proactive Agent — 主动型 AI 助手

## 定义

Proactive Agent 是指能主动感知用户意图、Always-on 了解用户上下文、具备长期记忆、并在合适时机自动采取行动的 AI 系统——而非被动等待用户提问的传统对话模型。

拾象判断：Proactive Agent 会成为 AI 时代的胜负手，是 2026 年最关键的不确定因素之一。

## 运作方式

**核心能力四要素**：
1. **意图识别**：Agent 可以在合适的时间 take actions，而非等待明确指令
2. **Always-on**：持续了解用户的实时 context（正在做什么、在哪里、和谁交互）
3. **长期记忆**：了解用户的长期目标和偏好，形成持续进化的用户模型
4. **主动行动**：在合适的时间主动推送信息、完成任务，而非被动响应

**OpenAI 的产品构想（Mark Chen 描述未来 ChatGPT）**：
- Memory 会被大幅升级
- 从用户问答交互中深层次学习、理解用户
- "提前准备"和用户有关的新问题、信息
- 需要新的硬件和交互承载（新设备形态）

**典型代表**：OpenAI Pulse、Thinking Machine Labs（个性化 Agent，新交互）

**Agent 吃掉谁的流量**：超级流量入口（Super App、Google Search）、手机厂商。豆包手机（ByteDance）是 Proactive Agent 抢流量的早期探索。

## 为什么重要

1. **新技术范式**：从"查询-回答"到"感知-行动"，是人机交互的范式级转变
2. **用户粘性**：Always-on + 长期记忆 = 无法替代的个性化体验，形成极强用户护城河
3. **真正的个性化**：当前 AI 每次对话都重置，Proactive Agent 能做到真正了解用户
4. **商业价值巨大**：从"工具"变成"助手"，订阅溢价空间大幅提升
