# 拾象AI洞察 全景概览

拾象科技（Shixiang Tech）是专注于 AGI 前沿的投资研究机构，每年发布 LP Meeting 报告，系统梳理全球 AI 竞争格局、技术趋势与二级市场投资策略。核心投资哲学：**只 bet 技术成长最陡峭的地方**。

## 摘要

2025年12月 LP Meeting 报告（"How to Play AI Beta"）提出7个 AI 关键判断：模型交替领先是常态、Google + OpenAI = $10T、2026年看到下一代范式信号（Continual Learning）、AGI 是现金流持久战、AI 必须回答商业价值问题、只 bet 技术最陡峭的地方、2026年 Proactive Agent 和多模态是投资主题。

**Ideal AGI Basket（2025年12月）**：

**头部 AI 公司 ARR（2025年12月）**：
- OpenAI：$21B；Anthropic：$9B；Claude Code / Cursor / Surge / Mercor：各 $1B

**ChatGPT vs Gemini 关键数据**：
- ChatGPT DAU/MAU：~25%，Gemini：~10%

- Google Search : ChatGPT 流量比：从95:5演变为85:15（2025年10月）
- ChatGPT 月留存率：从31%持续升至60%
