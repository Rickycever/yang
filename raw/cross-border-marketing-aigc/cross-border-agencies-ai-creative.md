## 二、工作流拆解(每家单独看)

### 2.1 蓝色光标:"All in AI" 战略级落地

**工作流架构(从战略到执行)**:

```
┌────────────────────────────────────────────────────────────────────┐
│ 战略层 (CEO 潘飞亲自抓)                                              │
│  - "All in AI" 战略                                                    │
│  - 95% 业务场景使用 AI                                                 │
│  - 70-80% 收入目标由 AI 驱动                                           │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ 平台层 (BlueAI 行业模型)                                               │
│  - 自研 + 合作: 智谱/微软/火山引擎/通义/Minimax/生数/Adobe              │
│  - 136 个 AI Agent(覆盖视频生成、智能投放等)                          │
│  - API Token 调用量 6000 亿次(Q3)                                      │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ 产品层 (蓝色多脑盒 + 心影 + 各种 Agent)                                  │
│  - 蓝色多脑盒: AI 创意厂牌(2025 CASE 100 入选)                        │
│  - 心影: 视频版本(数字人 3 分钟建模)                                    │
│  - Auto Slogan: 10 分钟生成品牌标语                                     │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ 业务层 (各事业部)                                                       │
│  - 整合营销: 95% 场景 AI 化,效率 +60-1000%                              │
│  - 数字广告: 国内 +35-100%                                              │
│  - 全球化出海: +100-1000%(BlueX Q4 同比 +4000%)                       │
└────────────────────────────────────────────────────────────────────┘
```

**创意团队日常工作流**(蓝色多脑盒 / 心影 团队口径):
1. **需求接入**: 客户 Brief 进入系统,Agent 自动拆分(品牌定位 / 受众 / 卖点 / 平台)
2. **创意发散**: 蓝色多脑盒(AI 创意厂牌)输出 50-100 个创意方向
3. **创意生产**: 设计师筛选 + AI 辅助(数字人/图生图/图生视频),单条素材成本降 30-90%
4. **批量生产**: API Token 6000 亿次调用,月产 5 万条创意,效率升 8-10 倍
5. **数据回流**: 投放表现回流到 BlueAI,持续微调行业模型

**AIGC 程度评估**:
- 战略级投入,CEO 亲自抓
- 数据资产深厚(10 亿+ 用户画像,1.8 亿+ 标注数据)
- 自研模型 + 多家外部模型混合使用,不押注单一供应商
- 已有 1000+ AI 客户案例,2025 目标 5-10 个 Agent 达到"专家级"产出
- 短板:Agent 单点能力(垂直深度)还在补

```
┌────────────────────────────────────────────────────────────────────┐

│  - AIEDA 方法论(诊断框架)                                              │
│  - 行业 know-how 沉淀(快消/餐饮/家居/鞋服/科技/电子/游戏/潮品)        │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ 模型层 (Google Cloud Gemini 2.5 / VEO3 / Imagen 4)                  │
│  - 全部基于 GCP,不押注单一国产模型                                      │
│  - 强依赖 Google AI 生态(广告主也在 Google 投)                          │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ 产品层 (Meet 三件套)                                                    │
│  - Meet Insight: AI 智能独立站诊断(品牌页面效率 +100%,转化 +25%,成本 -43%)│
│  - Meet Muse: AI 素材生成(文生图/多图参考/文生视频/图生视频)             │
│  - Meet Audience: AI 人群策略(种子组 CRV +14%,扩量组 ROAS +62%)       │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ 服务层 (三位一体)                                                       │
│  - AIDEA 咨询诊断                                                       │
│  - 建站电商 / 数据分析 / 用户运营 / 技术基建 四大服务                    │
│  - 长效运营陪跑                                                          │
└────────────────────────────────────────────────────────────────────┘
```

**创意团队日常工作流**(Meet Muse 口径):
1. **Brief 接入**: 客户输入商品 URL / 卖点 / 目标受众
2. **AI 诊断**: Meet Insight 评估当前独立站/落地页,识别优化点
3. **AI 素材生成**: Meet Muse 调用 Imagen 4 / VEO3,文生图 + 多图参考 + 文生视频 + 图生视频
4. **本地化适配**: 多语言 Prompt(中英日韩泰越印尼等),自动适配文化差异
5. **素材投放**: 接入 Google Ads / Meta / TikTok,实时追踪表现
6. **数据回流**: 表现数据回流到 Meet Audience 训练,持续优化人群策略

**关键案例数据**:
- 某滑雪品牌: 品牌页面设计效率 +100%,独立站转化率 +25%,网站优化成本 -43%
- 某服装品牌: 新款变体产出只需 **1 天**,新创意素材数量 +150%,视频质感"电影大片"
- Meet Audience: 种子组 CRV +14%,扩量组 ROAS +62%
- SEO 优化: 网站 SEO 健康分 +50%,AI 文案月均产出 200+ 篇 Blog,AI 率控制在 25% 以下

**AIGC 程度评估**:
- 业务级落地,产品矩阵跑通(Insight / Muse / Audience 三件套)
- 强依赖 Google AI 生态(Gemini / VEO3 / Imagen 4)
- 行业 know-how 深厚(10 年 + 100 万级用户数据)
- 短板:未公开自研模型,数据资产不如蓝标深厚

### 2.3 钛动科技:Tec-Creative AI 闭环

**工作流架构(从 AI 策划到垂直训练)**:

```
┌────────────────────────────────────────────────────────────────────┐
│ Step 1: AI 策划                                                         │
│  - 大数据 + AI 对话式策划案                                              │
│  - 市场分析 + 受众分析 + 卖点拆解                                        │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ Step 2: AI 创意生成(Tec-Creative 核心)                                 │
│  - 文生图 / 图生视频 / 视频编辑(智能混剪)                                │
│  - 批量模特换装、换品智能匹配                                             │
│  - TikTok 风格数字人(8 大文化区域,10+ 场景,6 种姿态,800+ 形象)            │
│  - 商品链接 → 一键生成数字人产品介绍视频                                  │
│  - 视频解构: 一键拆解爆款素材的脚本/标签/口播                             │
│  - 声音克隆 / 唇形对齐                                                   │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ Step 3: AI 智能投放                                                      │
│  - 批量投放 + 大规模策略组合                                              │
│  - Meta / Google / TikTok 全平台适配                                     │
└────────────────────────────────────────────────────────────────────┘
                              ↓
┌────────────────────────────────────────────────────────────────────┐
│ Step 4: 实时投放数据反馈 → Step 5: 垂直模型训练                          │
│  - 投放数据回流到 Navos 创意 Agent                                       │
│  - 训练垂直行业模型(电商 / 游戏 / 短剧)                                  │
└────────────────────────────────────────────────────────────────────┘
```

**创意团队日常工作流**(Tec-Creative 2.0 口径):
1. **AI 策划**: 输入商品 URL,自动提取页面信息,生成策划案
2. **AI 素材生成**: 三种典型场景
   - **快速上身图**: 平铺图 + 多款商品图 → 50+ 款上身效果图(某服饰黑五案例 2 天完成)
   - **图转视频**: 静态产品图 → 动态旋转/光影视频(降低测试成本)
   - **批量混剪**: 100 个原始素材 → 100 条差异化视频(0 积分免费,3 档混剪模式)
3. **数字人**: 1000 字口播文案 + 性别/年龄/区域/场景/姿态/服装参数 → TikTok 风格数字人视频
4. **爆款解构**: 上传爆款视频 → 自动拆解脚本/分镜/标签/口播 → 复刻
5. **数据回流**: 投放数据 → Navos Agent 训练 → 下一轮创意优化

**关键能力数据**:
- 批量混剪: **0 积分免费** 生成 100 条视频(2025-06 上线)
- 数字人: 800+ 形象库,8 大文化区域(东亚/东南亚/东欧/西欧/中东/北美/拉美/非洲)
- AI 实验室: 声音克隆、图片生成数字人、唇形对齐
- 案例: 某服饰品牌黑五 2 天产出 50+ 款展示素材
- 案例: 测投素材(白底图) → 图生视频(动态) → 实拍(高质量) 三段式降低测试成本

**AIGC 程度评估**:
- 工具级深耕,创意环节渗透最深(批量混剪 / 数字人 / 爆款解构 三个杀手级功能)
- 全链路闭环:策划 → 创作 → 投放 → 数据回流 → 垂直训练
- 行业垂直: 电商 / 游戏 / 短剧 三大出海热门
- 短板:平台型公司,不做品牌服务,客户粘性靠工具效率

## 四、对自建工具的启示

如果你的目标是"自建 AIGC 创意生成和分发工具",这三家公司的策略值得重点参考:

### 4.1 不要试图复制蓝标的"集团 AI 转型"路径

蓝标走的是"All in AI 战略 + 数据飞轮 + Agent 矩阵",这要求公司体量大(600 亿营收)+ 数据厚(10 亿+ 用户画像)+ 自研深(BlueAI)。**普通公司走不通**——蓝标 2024 净利润亏损 3-4 亿(剔除一次性因素后 +2~3 亿),就是因为转型阵痛大。**你的工具应该走"专精路线",不要走"全能路线"**。

### 4.2 重点学习飞书深诺的"AIEDA 方法论"和 Meet 三件套产品形态

飞书深诺的产品形态(Insight 诊断 / Muse 生成 / Audience 投放)是行业最清晰的,值得直接参考:
- **Insight 工具**:不直接做创意,但帮广告主做"投放前诊断"。这一层国内几乎空白,工具型公司可切入
- **Muse 工具**:AI 素材生成,核心是"行业模板 + 品牌一致性"
- **Audience 工具**:AI 人群策略,本质是"种子人群构建 + 扩量优化"

如果自建工具,可以从 **"Insight + Muse"** 两件套切入,Audience 层用 Meta Marketing API + 第三方数据组合。

### 4.3 重点学习钛动的"创意 Agent"和"批量混剪"功能

钛动的 Tec-Creative 在创意环节做得最透:
- **批量混剪**:100 个素材 → 100 条视频,0 积分免费。这是 Meta Advantage+ Creative + Catalog 的"中国版",但做了"中文出海 + 多文化区域适配"
- **TikTok 风格数字人**:8 大文化区域 + 10+ 场景 + 6 种姿态 + 800+ 形象库。这是国内最深的数字人素材库
- **爆款解构**:上传爆款视频 → 自动拆解脚本/分镜/标签/口播 → 复刻。这是 Meta 短期内不会做的"逆向工程"能力

如果自建工具,**数字人 + 爆款解构**是两个最值得深挖的差异化点。

### 4.4 避开"重复造轮子"的红海

三家公司已经把以下能力做到成熟,自建工具不要再做:
- ✅ 基础图片生成(文生图 / 图生图)
- ✅ 基础视频生成(图生视频)
- ✅ 基础数字人(单一区域 / 单一场景)
- ✅ 基础 AI 文案(通义 / DeepSeek / Gemini 都能做)

应该做**Meta 自己没做 + 三家公司没覆盖 + 中型公司付得起**的中间地带:
- 行业模板(美妆/食品/3C/汽车/教育/医美) — Meta 没做,蓝标/飞书/钛动做了但要绑定
- 跨渠道编排(Meta + TikTok + 巨量) — 三家公司各有偏向,但没有"全渠道中立工具"
- 创意特征回流训练 — 蓝标/飞书在内部用,不对外开放;钛动有 Navos Agent 但只服务自家客户

### 4.5 创意团队人员配置建议(对标)

参考三家公司,自建工具团队的创意侧建议配置(10-15 人):
- **AI 创意总监**(1 人):懂创意 + 懂 AI,统筹方向
- **Prompt 工程师**(2-3 人):行业模板 + 品牌一致性 + 数字人 Prompt
- **AI 美术**(3-4 人):Midjourney / Stable Diffusion / Sora / 可灵的资深用户,会做参数调优
- **传统美术**(2-3 人):负责高精素材 + 品牌资产维护
- **数据分析师**(1-2 人):创意表现 → 训练样本标注 + 飞轮数据回流
- **投放运营**(2-3 人):Meta Marketing API + CAPI 接入,AB 实验执行
