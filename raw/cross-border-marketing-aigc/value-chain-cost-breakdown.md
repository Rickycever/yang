## 一、顶层:全行业营销预算结构(2025-2026)

### 1.1 总盘子有多大

- **2025 全球广告市场(不含美国政治广告):1.14 万亿美元**,+8.8% YoY [WPP Media 2025]
- **2025 中国出海营销市场:420 亿美元(2024)→ 1237 亿美元(2029),CAGR 24.3%** [东方财富 2025 报告]

- **2025 美国创作者广告支出:371 亿美元(+26% YoY,是媒体行业增速的 4 倍)** [IAB]
- **2025 全球电商广告支出:1782 亿美元,首次超过电视广告** [WPP Media]

### 1.2 营销预算的"三层漏斗"

对一个中型出海企业(假设年营收 1-10 亿),营销预算三层拆解:

```
┌────────────────────────────────────────────────────────┐
│ Layer 1: 营销总预算(占营收 12-25%)                          │
│  - 北美: 15.3% 营收(Statista 2023)                         │
│  - 东南亚: 9.7% 营收                                        │
│  - 全球跨境电商平均: 12.8% 营收                              │
│  例: 年营收 5 亿 → 营销预算 6400 万 - 1.25 亿              │
└────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────┐
│ Layer 2: 数字营销预算(占营销预算 60%+)                      │
│  - 数字广告占 60%+ 全球广告(WPP 2025)                       │
│  - 出海主推数字渠道(海外传统媒体几乎不做)                    │
│  例: 营销预算 8000 万 → 数字营销 5000 万+                   │
└────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────┐
│ Layer 3: 创意+工具+代理(占数字营销预算 30-50%)             │
│  - 创意制作 10-15%                                          │
│  - 代理/服务费 10-20%                                       │
│  - 工具/技术 2-5%                                            │
│  - 归因/测量 1-3%                                           │
│  例: 数字营销 5000 万 → 创意+工具+代理 1500-2500 万         │
└────────────────────────────────────────────────────────┘
```

**关键判断**:**创意生产 + 工具/技术 = 数字营销预算的 12-20%**,这部分是 AIGC 创意工具最直接的市场空间。

## 四、价值链分布分析(谁拿走了哪些钱)

### 4.1 完整的价值链结构

```
广告主(品牌方)
  │
  ├──[直接成本]──> 媒体采买(Meta/Google/TikTok 等):60-70% 数字营销预算
  │                ↓
  │              用户触达
  │
  ├──[代理/服务]──> 出海营销代理(蓝标/飞书深诺/钛动):10-20% 数字营销预算
  │                  ↓
  │                创意生产 + 投放优化 + 客户对接
  │
  ├──[工具/技术]──> MarTech SaaS(HeyGen/AppsFlyer/AdCreative.ai):2-5% 数字营销预算
  │                  ↓
  │                AIGC 创意 / 归因 / 优化工具
  │
  ├──[创意生产]──> 内部团队 + 外包:10-15% 数字营销预算
  │                  ↓
  │                文案 / 视觉 / 视频 / 数字人
  │
  └──[测量/合规]──> 归因工具 / 法律咨询:1-3% 数字营销预算
                     ↓
                   归因数据 / 合规审核
```

### 4.2 各层级的"价值捕获能力"

| 层级 | 占比 | 价值捕获能力 | 议价能力 | 毛利率 |
|---|---|---|---|---|
| **媒体平台(Meta/Google)** | 60-70% | 极强(网络效应 + 用户数据) | 极强 | 80%+ (Meta 2024 营业毛利率 82.5%) |
| **出海营销代理(蓝标/飞书/钛动)** | 10-20% | 中(规模 + 资源) | 中(客户可绕过) | 15-20% (蓝标 2024) |
| **MarTech 工具(HeyGen/AppsFlyer)** | 2-5% | 强(技术壁垒) | 中-强 | 60-80% (SaaS 模型) |
| **创意生产团队** | 10-15% | 中(创意是稀缺品) | 弱-中 | 30-50% (人力密集) |
| **测量/合规** | 1-3% | 弱(标准服务) | 弱 | 20-30% |

**关键判断**:
- **媒体平台拿走了 60-70% 的钱**(高议价)
- **代理拿走 10-20%**(可被替代,议价弱)
- **MarTech 工具拿走 2-5%**(技术壁垒强,毛利率最高 60-80%)
- **创意生产 10-15%**(议价中等,但 AIGC 在改造这一层)

### 4.3 AIGC 工具的"夹缝机会"

AIGC 创意工具不直接和媒体平台(60-70%)竞争,也不和代理(10-20%)全面竞争,而是**直接改造创意生产这一层(10-15%)**:

```
价值转移方向:

传统价值链:
  广告主 → 代理 → 创意团队(高成本)→ 投放

AIGC 改造后:
  广告主 → AIGC 工具(自建 or SaaS) → 投放
              ↓
            (代理被压缩,只保留服务费)
```

**核心判断**:
- AIGC 工具的 target market = **创意生产这一层 10-15% × 数字营销预算 60-70% × 总营销预算 12-25% = 数字营销预算的 6-10%**
- 例:全球数字广告市场 1.14 万亿美元 × 6-10% = **684-1140 亿美元/年 是 AIGC 创意工具的潜在市场空间**
- 蓝标 2024 AI 驱动收入 12 亿元 / 目标 30-50 亿 / 百亿,说明**头部玩家已能切到 100 亿+ 的市场**

### 4.4 Meta vs Google 价值链对比

| 维度 | Meta | Google |
|---|---|---|
| **媒体采买占比** | 30-40% 出海营销预算 | 20-25% |
| **CPC 区间** | $0.45-$3.77(行业差异大) | $0.5-$50(关键词差异极大) |
| **创意内容侧重** | 视觉(图片 + 视频 + Reels) | 文本(搜索 + 购物 + YouTube) |
| **创意 API 能力** | Marketing API v21+ 强(creative / ad / asset_feed_spec) | Google Ads API + PMax 资产管理 |
| **AIGC 原生能力** | Advantage+ Creative(免费基础) + Movie Gen 测试中 | Gemini 3 / Veo 3.1 / Imagen 4 / Pomelli |
| **数据闭环** | CAPI + 增量归因(强) | GA4 + 转化跟踪(部分被 cookie 限制削弱) |
| **代理依赖** | 较高(创意门槛高) | 较低(自动化出价成熟) |
| **创意生产节点价值** | **更高**(AIGC 创意差异大) | **较低**(创意 + 出价自动化) |

**关键判断**:**Meta 侧的 AIGC 创意工具机会 > Google 侧**。原因:
- Meta 的 Advantage+ Creative 已经"免费化"基础能力,但 Andromeda 抬升了创意权重 → AIGC 创意空间更大
- Google 自动化出价成熟,创意差异化空间被 Gemini 系列压缩
- Meta 用户基数大、视觉平台对创意要求高、AIGC 工具能直接做 DPA 目录变体

## 五、AIGC 创意工具在价值链中的"卡位策略"

### 5.1 三种卡位模式

**模式 1:做"创意生产节点"的 SaaS(对标 AdCreative.ai / HeyGen)**
- 角色:直接服务广告主/代理的创意工具
- 收费:订阅 + 按量(月费 $29 起,按生成量计费)
- 优势:轻资产,可快速规模化
- 挑战:工具同质化严重,容易被替代

**模式 2:做"创意生产 + 投放分发"的中台(对标蓝标 BlueAI)**
- 角色:从创意到投放的全链路自动化
- 收费:效果分成 + 服务费
- 优势:飞轮数据闭环,客户粘性高
- 挑战:重资产,需要长期投入

**模式 3:做"垂直行业创意模板 + SaaS"(对标钛动 Tec-Creative)**
- 角色:聚焦电商/游戏/短剧等垂直行业
- 收费:订阅 + 模板市场分成
- 优势:行业 know-how 深,差异化高
- 挑战:行业天花板有限

**对自建工具的推荐**:
- 中型互联网公司内部投放团队 → **模式 1 + 模式 3**(先用 SaaS 跑业务,再沉淀行业模板)
- 代理公司(蓝标/飞书/钛动) → **模式 2**(全链路自动化,作为甲方内部的工具)
- 行业 SaaS(美妆/3C/汽车) → **模式 3**(垂直行业 know-how)

### 5.2 Meta vs Google 工具投入分配建议

| 投入方向 | Meta 侧(占比 70%) | Google 侧(占比 30%) |
|---|---|---|
| 创意生成 | 数字人 + 图生视频 + DPA 目录变体(主) | 文本生成 + YouTube 视频脚本(次) |
| 创意分发 | Marketing API v21+ 深度对接 | Google Ads API + PMax 资产管理 |
| 创意训练 | Meta 创意表现 → 内部模型(独家护城河) | Google 数据公开度高,训练价值小 |
| 优先级 | **高** | 中 |

## 六、ROI 反推:自建工具的"经济账"

### 6.1 场景假设

中型互联网公司出海投放团队:
- 年数字营销预算:**5000 万**
- 创意制作预算:**750 万(15%)**
- 创意生产痛点:产能跟不上,3 天大促要出 50 条素材,真人拍摄根本来不及

### 6.2 自建 AIGC 工具的 ROI 测算

**投入侧**:
- 工具/技术:200 万/年(MVP + 集成)
- 团队人力增量:200 万/年(+3 个 AI 美术 + 1 个 Prompt 工程师)
- 培训 + 试错:50 万/年
- **总投入:450 万/年**

**产出侧**:
- 创意制作成本降低:750 万 × 50% = 375 万(从 750 万降至 375 万)
- 创意产能提升带来的增量 GMV:难精确测算,假设**+10% 转化率提升 × 5 亿 GMV × 5% 净利率 = 250 万净利润增量**
- 减少外包依赖:100 万/年
- **总产出:725 万/年**

**ROI** = 725 / 450 = **1.61**

**回本周期**:约 7-8 个月

**判断**:**自建工具是经济可行的**,但需要 1-2 年的投入周期才能跑通。

### 6.3 集成第三方(不自建)的 ROI 对比

**投入侧**:
- 工具订阅:50-100 万/年(HeyGen 团队版 + 硅基 + 剪映 + AppsFlyer)
- 团队人力:**不变**(原班人马)
- **总投入:100 万/年**

**产出侧**:
- 创意制作成本降低:750 万 × 30% = 225 万(削减 30%)
- 创意产能提升带来的增量 GMV:难精确,假设**+3% 转化率 × 5 亿 GMV × 5% 净利率 = 75 万**
- 减少外包:50 万/年
- **总产出:350 万/年**

**ROI** = 350 / 100 = **3.5**

**回本周期**:约 3-4 个月

**判断**:**集成第三方的短期 ROI 更高(3.5 vs 1.61),但自建长期护城河更深**。建议路径:
- **第 0-12 月**:集成为主,跑通业务
- **第 12-18 月**:评估自建 ROI,聚焦最高 ROI 场景(数字人 + DPA 目录变体)自建
- **第 18 月+**:逐步把高 ROI 场景自建,降低单位成本

- Meta 创意生产节点占 10-15% 数字营销预算,Google 同节点价值更小(自动化出价压缩差异)

**3. 自建 vs 集成的决策**:
- **集成第三方 ROI = 3.5(回本 3-4 月)**,适合早期快速跑通
- **自建 ROI = 1.61(回本 7-8 月)**,适合长期护城河
- **推荐路径**:集成 0-12 月 → 评估自建 → 自建 12 月+

**4. 创意工具的卡位策略**:
- 模式 1(SaaS):轻资产可快速规模化,但同质化
- 模式 2(中台):重资产但客户粘性高(对标蓝标 BlueAI)
- 模式 3(垂直行业):差异化高,行业 know-how 强(对标钛动 Tec-Creative)

**5. 价值链变化方向**:
- 媒体平台(60-70%)→ 持续拿走大部分价值
- 代理(10-20%)→ 被 AIGC 工具压缩,只保留服务费
- MarTech 工具(2-5%)→ 价值扩大(从创意到归因到训练)
- **创意生产(10-15%)→ 是 AIGC 工具的主战场**
- 测量/合规(1-3%)→ 稳定

## 一句话总结

**Meta 侧创意生产的"夹缝"是 AIGC 工具的主战场**:全球 684-1140 亿美元/年的市场空间,中型互联网公司自建工具 ROI 1.61(7-8 月回本),集成 ROI 3.5(3-4 月回本)。**推荐先用集成跑通业务,12 个月后再聚焦最高 ROI 场景(数字人 + DPA 目录变体)自建**。
